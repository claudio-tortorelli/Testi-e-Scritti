<h2><a name="2"></a>Pane e Salame</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<h3><a name="3"></a>I confini di un esilio - 03/03/2006</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Lontane luci evocano la solitudine <br/> 	nel buio. <br/> 	Metalliche galassie di stelle artificiali <br/> 	le raffinerie. <br/> 	Questa terra sconsacrata <br/> 	non so disprezzare. <br/> 	Essa è uno scrigno <br/> 	con dentro la chiave.</p>
<p class="indent">
Quando si immagina la definizione di un confine <br/> 	si ignora quanto le genti e la terra <br/> 	che di presso vi cadono <br/> 	siano sì poco definiti. <br/> 	Paradosso. <br/> 	Quei luoghi pregni di magia <br/> 	dove la terra si inabissa nel mare <br/> 	dove il monte separa le valli <br/> 	mi infondono la curiosità del mistero <br/> 	che nasce dal percorrere <br/> 	l'assenza di limiti concreti <br/> 	che attraversa il continuo.</p>
<p class="indent">
Ma qui, in pianura, ogni cosa  <br/> 	dannatamente omogenea <br/> 	talmente accessibile <br/> 	si perde. <br/> 	Si isola. <br/> 	Nessun confine mi separa da lei <br/> 	solo qualche chilometro <br/> 	che di nuovo <br/> 	non posso percorrere.</p>
<p>
Lontane luci si spengono alle mie spalle <br/> 	nella sua notte.</p>
<h3><a name="4"></a>Informe - 01/03/2006</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Io non sono <br/> 	quel che appaio <br/> 	e mi muovo <br/> 	silenzioso <br/> 	in questa casa di specchi <br/> 	e riflessi. <br/> 	Concave deformità mi seguono <br/> 	e poi dileguano. <br/> 	Sgomento subentrante. <br/> 	Di là si chiude un sipario di paura <br/> 	scende pesante <br/> 	la vergogna delle convinzioni. <br/> 	Biologiche necessità meccaniche <br/> 	conducono a orgogliose convenzioni <br/> 	che minano il prato di margherite <br/> 	dove eravamo scalzi.</p>
<p class="indent">
Un giorno tu mi vedrai, <br/> 	senza forma. <br/> 	Per te saranno chiare le mie sembianze <br/> 	dissolte nel tempo. <br/> 	E il mio silenzioso dolore di oggi <br/> 	splenderà caldo e innocuo <br/> 	come ieri il tuo seno. <br/> 	Allora saranno le incomprensioni, le incomprese. <br/> 	La tua mano trarrà dal mio cuore sterco e rifiuti. <br/> 	Seccheranno dimenticati al sole... <br/> 	un sole diverso, un sole tranquillo, <br/> 	che ancora rinasce dopo ogni nuvola.</p>
<p class="indent">
Quell'attimo eterno di Olimpo <br/> 	soffia e profuma come il vento di marzo <br/> 	nella mia mente e sulle povere cose <br/> 	che attendono il color della sera <br/> 	nel quieto tepore.</p>
<p>
Quando non avrò più immagine, <br/> 	né ombra...</p>
<h3><a name="5"></a>Pensando di partire per il Messico - 18/02/2006</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Mi appoggio leggero <br/> 	a quella parte di me <br/> 	che riposa <br/> 	respira lontano. <br/> 	Ogni suo sogno sorge nel mio cuore <br/> 	e nessuno ne trattengo. <br/> 	Ogni sua lacrima scivola calda <br/> 	sulla mia guancia.</p>
<p>
Tornerai </p>
<p class="indent">
ti dico <br/> 	a farti materia <br/> 	staccandoti nuovamente inviolata <br/> 	dalla memoria.</p>
<p class="indent">
Quando, quando il senso <br/> 	sarà perduto <br/> 	e il rimorso di non conoscerti <br/> 	sarà svanito.</p>
<p class="indent">
Faville; si spengono nel caminetto <br/> 	le braci dei quando e dei perché. <br/> 	Ora fa freddo ma  <br/> 	ciò che mi sfugge  <br/> 	con lei tornerà. <br/> 	Aspetterò un attimo solo  <br/> 	quell'attimo.</p>
<h3><a name="6"></a>Uomo di poca fede - 08/01/2006</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Non sei qui, seduta <br/> 	sul letto disfatto <br/> 	con me. <br/> 	Senza sforzo immagino ancora <br/> 	il colore dei tuoi occhi <br/> 	e il tuo buffo modo di dormire. <br/> 	O il tuo nome. <br/> 	Ecco che ora ti alzi per mostrarmi il taglio <br/> 	del tuo nuovo vestito <br/> 	e mi chiedi... <br/> 	ma non posso risponderti. <br/> 	Se tu non esisti.</p>
<p class="indent">
Tutto il resto posso creare io per te <br/> 	ma non la reale presenza del tuo spirito <br/> 	in questa stanza, la tua volontà di esserci <br/> 	che ogni palpabile immagine schizza.</p>
<p class="indent">
Gioco a carte col mio futuro: <br/> 	aspetto il jolly che rompa il ritmo, <br/> 	ma quel mazzo di carte, truccate, <br/> 	insinua un dubbio costante. <br/> 	Mille code che scompaiono <br/> 	dietro l'angolo.</p>
<p class="indent">
Conosco l'argilla di cui sei fatta; <br/> 	della stessa son fatto io. <br/> 	Traggo  <br/> 	dalla luce delle stelle <br/> 	e dai grigi pomeriggi di qualche gennaio <br/> 	la forma del tuo vaso. <br/> 	E sapessi che qualcuno altrove t'ha plasmata <br/> 	ti cercherei <br/> 	per trovare il tuo Creatore. <br/> 	Se ci credessi, ti troverei <br/> 	ma devo trovarti per crederti.</p>
<p class="indent">
Senza fine  <br/> 	l'attesa di unirmi a te <br/> 	in un nuovo impasto <br/> 	sarebbe sopportabile. <br/> 	Il mio futuro giace addormentato <br/> 	al posto tuo <br/> 	in questo letto disfatto. <br/> 	In un albero abbattuto <br/> 	la vita prosegue distinta <br/> 	nel fusto già adorno <br/> 	come nel nero-invisibile <br/> 	delle radici.</p>
<h3><a name="7"></a>Statue di Creta - 18/12/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Alcune parole <br/> 	son scritte su soffitti di carta velina. <br/> 	Altre mi circondano <br/> 	su pareti di carta di riso <br/> 	oppure calpestate <br/> 	giacciono nei pavimenti di carta crespa.</p>
<p class="indent">
Cubi di parole <br/> 	mi contengono <br/> 	e mi tornano a parlare di gente di pietra <br/> 	che un giorno le scrisse. <br/> 	Quanto è fragile <br/> 	quanto è stretto alle volte <br/> 	il mio cubo di carta. <br/> 	Luce, ombra, acqua e fango <br/> 	lo sconvolgono, commuovono e deformano. <br/> 	Così stesse parole <br/> 	tanti diversi significati. <br/> 	Così le statue tornano a muoversi <br/> 	si pongono a mia discrezione nella prospettiva <br/> 	delle loro nuove inflessioni.</p>
<p class="indent">
Cammino passeggio in un giardino di statue di creta, <br/> 	ogni abbraccio plasma un'incomprensione, <br/> 	ogni carezza immaginaria rimane <br/> 	su una pelle distante.</p>
<h3><a name="8"></a>Cesenatico - 12/12/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p>
Pascoli erbosi <br/> 	senza greggi da pascere.</p>
<p class="indent">
Dove va l'acqua che scorre <br/> 	nel fiume la notte... <br/> 	come confluisce, schivando secche baciando rocce <br/> 	per chi gorgoglia questo savio tormento?</p>
<p class="indent">
Dei passi gelidi, arido rumore su ponti inutili. <br/> 	V'è un solstizio di vita <br/> 	nell'oscuro momento di questo canale <br/> 	si eclissa fluttuando tra la noia e la malinconia <br/> 	un peschereccio.</p>
<p class="indent">
Tante luci <br/> 	spente <br/> 	tanta gente <br/> 	altrove. <br/> 	Nel silenzio odo me stesso <br/> 	parlarmi domani.</p>
<h3><a name="9"></a>Passanti - 30/10/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p>
(alle passanti, che sanno di passarmi dentro)</p>
<p class="indent">
Sono le belle passanti <br/> 	fatte di nebbia <br/> 	riflesso furtivo di sguardi <br/> 	intensi istanti distanti <br/> 	e mete così lontane: <br/> 	un portone tra due parole <br/> 	solo pensate. <br/> 	Sono loro  <br/> 	attimo mai colto <br/> 	che spiegano passando <br/> 	magici tappeti su cui volare <br/> 	dentro questi stessi luoghi. <br/> 	Affacciarsi sul vecchio vicolo <br/> 	da pertugi nuovi e ora reali; <br/> 	spalancare ogni marcio uscio! <br/> 	per Dio ridere, ridere insieme! <br/> 	Con l'urgenza del bimbo e dell'attore <br/> 	cercare una mano, stringerla, baciarla. <br/> 	E se una spinta, una bottiglia o i fari di un'auto <br/> 	quella mano fan sfuggire, <br/> 	non solo cenere rimane,  <br/> 	non solo acqua nel mio alambicco.</p>
<p class="indent">
Brilla un po' in tasca la scintilla, <br/> 	tre passi di tango <br/> 	la via consueta <br/> 	incantano.</p>
<p>
Se mi voltassi, tutto sarebbe passato.</p>
<h3><a name="10"></a>Seno - 03/10/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p>
(agli angeli)</p>
<p class="indent">
Nulla compete con ciò che si può indovinare, <br/> 	ciò che si ha e si ignora di avere <br/> 	sotto vesti, veli o lenzuola <br/> 	si plasma l'estro che forgia l'attrazione  <br/> 	e infine soggioga il virile distacco <br/> 	rendendo l'offesa esuberante nella tenzone <br/> 	difesa di una tenera unione.</p>
<p class="indent">
Nessuna forma e forma perfetta <br/> 	ha il porto di ogni marinaio, <br/> 	incoscienti naufraghi approdano alle rive <br/> 	ormai persi nel mirabile paradosso <br/> 	di un abbraccio ciclico <br/> 	che li avvolge nel tempo <br/> 	mentre l'emblema della creazione <br/> 	sempre presente <br/> 	il tempo dissacra.</p>
<p class="indent">
Fortunati coloro che sanno dove baciare, <br/> 	che sanno bagnarsi dove le rocce si sciolgono in mare. <br/> 	E' l'umiltà l'arma del trionfo, <br/> 	farsi trascinare senza domandarsi, <br/> 	rendere le armi curiosi della sconfitta <br/> 	affascinati dalla malizia infantile <br/> 	che si riflette negli angeli la sera. <br/> 	Si porge una guancia al candore <br/> 	si appoggia l'altra al calore <br/> 	e più non v'è schiaffo che possa far male.</p>
<p>
Un angelo mi ha regalato  <br/> 	ciò che non può essere reso.</p>
<p class="indent">
Un giorno scoprii casualmente la mia condizione di vedovanza <br/> 	e un sorriso amaro salì <br/> 	in un rigurgito di mancanza.</p>
<h3><a name="11"></a>Fine secolo - 29/09/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p>
(Ancora autunno)</p>
<p class="indent">
E' come una piccola vertigine <br/> 	sul bordo esterno della coscienza <br/> 	strapiombo, una farfalla sfrutta il vento <br/> 	mentre uno sguardo stanco <br/> 	intorno <br/> 	appesantisce le mie ali.</p>
<p class="indent">
Mi sto aprendo ad una fredda fretta fatale <br/> 	di vivere, forse casualmente, <br/> 	dissanguando i miei giorni <br/> 	forse volutamente, <br/> 	sono parassita di me stesso. <br/> 	Quando creo il mio futuro, <br/> 	e poi vi credo.</p>
<p class="indent">
Se ne vanno <br/> 	tutti lontano <br/> 	e non recano ricordo <br/> 	di me <br/> 	i miei giorni. <br/> 	Con loro tanti compagni <br/> 	tante compagne. <br/> 	Così ripeto convinto <br/> 	le cose da me scritte <br/> 	prima di nascere. <br/> 	Solo nel mio percorso tra gli altri.</p>
<p class="indent">
Perché il mio tempo è sbagliato: <br/> 	sono figlio del passato prossimo <br/> 	e del futuro anteriore. <br/> 	Come tutti in sella ad un cavallo immaginario <br/> 	spero che le mie gambe mi portino altrove <br/> 	al di là di questo secolo.</p>
<h3><a name="12"></a>Cose da dire, cose da fare - 19/09/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Cose da dire <br/> 	cose da fare. <br/> 	Ho scavato l'alveo di un fiume <br/> 	ed ora aspetto la pioggia. <br/> 	Se ripenso a quante volte <br/> 	m'è venuto da vivere, <br/> 	se ripenso a quante volte <br/> 	ho vissuto l'autunno <br/> 	scopro nuove profondità <br/> 	dove rimbalza l'eco monatto <br/> 	di pallide giovani mani, <br/> 	le carezze di una donna che ama.</p>
<p>
Nulla più soffice <br/> 	al pensiero <br/> 	al desiderio.</p>
<p class="indent">
Ogni cosa ho già detto <br/> 	in altri mondi <br/> 	ogni cosa ho già fatto. <br/> 	La distanza ignorante <br/> 	è l'ombra di un cipresso  <br/> 	meridiana e sestante <br/> 	al tramonto s'allunga <br/> 	s'infila dentro le crepe di un muro <br/> 	le allarga <br/> 	poi si fonde alla notte, <br/> 	tacendo  <br/> 	i significati del non essere <br/> 	che trascendono i simboli.</p>
<p class="indent">
Prima che tramonti il sole <br/> 	cose da dire <br/> 	cose da fare <br/> 	come ieri <br/> 	mai più.</p>
<h3><a name="13"></a>Dirompente impotenza bollente - 30/07/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Nei meriggi di deriva <br/> 	se le tortore cantilenano la calura <br/> 	ricorre la solita paura <br/> 	cinge il torace una stretta cintura <br/> 	e cola sul letto una luce ovattata <br/> 	dirompente riflesso del fuoco esterno.</p>
<p>
Pavide mani <br/> 	afferrano ombre di attimi <br/> 	malvissuti.</p>
<p class="indent">
Perfetta metafora del mio voler essere: <br/> 	il sole <br/> 	e una stanza in penombra; <br/> 	l'immobile rovente <br/> 	e l'incapace, impotente <br/> 	voglia di volare.</p>

<h3><a name="15"></a>Compagna di viaggio - 22/02/2006</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Il sole di questo primo <br/> 	freddo mattino <br/> 	risplende nei fossi <br/> 	d'ambra lucenti. <br/> 	Un uccello segue il corso dei binari <br/> 	come un delfino la nave. <br/> 	E' una ragazza, è il richiamo <br/> 	di una foresta. <br/> 	Lunghi capelli castani. <br/> 	Arriva la stazione. <br/> 	Scompare la notte. <br/> 	Scende la grazia semplicemente <br/> 	un sorriso. <br/> 	Rimango a fissare il ricordo <br/> 	riflesso <br/> 	sul finestrino. <br/> 	Compagna <br/> 	di un altro viaggio. <br/> 	Per un po' satellite <br/> 	di una Venere inviolabile.</p>
<h3><a name="16"></a>Dormire oltre il giusto - Febbraio 2006</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Penombra e silenzio. <br/> 	Un accenno crepuscolare al mal di testa. <br/> 	Apro le persiane in un surreale riflesso <br/> 	di luce già pomeridiana. <br/> 	Non me l'aspettavo, <br/> 	non mi son reso conto <br/> 	di tanta lunga incoscienza. <br/> 	Mi soffermo sul cielo azzurro cielo <br/> 	di febbraio. <br/> 	Un cielo così piccolo e così grande <br/> 	che contiene a fatica se stesso <br/> 	infinite volte <br/> 	e sotto di lui un po' di Bologna <br/> 	un po' di Firenze, Roma, Brindisi, Ferrara <br/> 	nei palazzi di Arezzo. <br/> 	Passa una bianca nuvola veloce e bassa nel vento <br/> 	e si ferma la ricorsione. <br/> 	Dove son stato questa mattina <br/> 	oltre che a letto?</p>
<h3><a name="17"></a>Case e pensieri - Ottobre 2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ho vissuto in case <br/> 	dove ho respirato i pensieri <br/> 	associati dagli abitanti <br/> 	alle loro situazioni.</p>
<p class="indent">
Case nelle cui stanze le cose <br/> 	identificano i riflessi d'umore <br/> 	nel tempo passato <br/> 	di chi li ha colte <br/> 	in uno sguardo d'insieme <br/> 	o nell'attenzione assente <br/> 	com'è la mia.</p>
<p class="indent">
Il mio mondo <br/> 	si fa mattonella, <br/> 	il mio occhio <br/> 	è lente che concentra <br/> 	fasci di input variopinti <br/> 	sui lembi più distratti <br/> 	del cervello.</p>
<p class="indent">
Ascolto altrui passioni <br/> 	riferite da una tenda trapassata di luce.</p>
<p class="indent">
Attimi e attimi di silenzio <br/> 	parleranno domani di me <br/> 	e del mio chiasso.</p>
<h3><a name="18"></a>Ottobre in Orte - Ottobre 2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ottobre <br/> 	è nell'umido degli orti <br/> 	ciaf ciaf dei miei passi corti; <br/> 	odore forte <br/> 	di vita e di morte <br/> 	traversando questa corte <br/> 	odo distendersi campane distorte <br/> 	al di là dei muri, oltre le porte <br/> 	una bolla di tempo, un catarro di sorte.</p>
<h3><a name="19"></a>Stereogrammi calabresi - Agosto 2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Un lunga piroetta <br/> 	che allontana quel che mi segue <br/> 	ovunque. <br/> 	Metri in musica si contano <br/> 	finché non ci si dissolve nel sud <br/> 	un altro <br/> 	modo finalmente influente di fluire <br/> 	infine <br/> 	il mare.</p>
<p class="indent">
Rifletto <br/> 	sull'originalità dei miei pensieri <br/> 	sulla capacità di appropriarmi di parole <br/> 	mai dette. <br/> 	Le mie mani pazienti filtrano granelli. <br/> 	Cercano frammenti mai soppesati <br/> 	ma loro scorrono indolenti <br/> 	e diventano nulla <br/> 	di diverso. <br/> 	Il senso si ritira come marea <br/> 	quando l'intuizione scolpisce petali di pietra, <br/> 	tanti fiori prigioni, mai sbocciati <br/> 	così profumati. <br/> 	E' notte, in spiaggia falò e lampi di stelle.</p>
<p class="indent">
Tendo allora le braccia al mare <br/> 	e il mare le colma di pietre. <br/> 	Ad ogni onda <br/> 	la meraviglia di veder assortire in grembo <br/> 	un arcobaleno litico <br/> 	che il destino mi inventa <br/> 	ma che io permetto. <br/> 	Il mio potere è grande quanto la nostalgia di saperlo già perduto <br/> 	tra poco, una notte di inverno <br/> 	mentre il mare sarà ancora li <br/> 	quando avrò già dimenticato.</p>
<p class="indent">
Bagnanti riflessi <br/> 	sale sole <br/> 	case feste fiumare  <br/> 	di gente <br/> 	il silenzio della moltitudine <br/> 	che d'estate si compie <br/> 	nello sforzo di indovinare <br/> 	un volto nascosto. <br/> 	Il mio.</p>
<h3><a name="20"></a>Terrore e incomunicabilità - 25/07/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Vedo piovere sassi <br/> 	dall'altro lato di un muro. <br/> 	Ogni sasso, sul sasso precedente. <br/> 	Sotto anime sepolte. <br/> 	Da ogni mucchio escono fuochi <br/> 	e strani fiori <br/> 	perduti <br/> 	che Nessuno coglie. <br/> 	Nessuno aggira il muro <br/> 	ma rimane solo e non vede <br/> 	chi tira i sassi, <br/> 	perché ora <br/> 	è dall'altra parte.</p>
<h3><a name="21"></a>Fatine - 22/07/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Hanno occhi smeraldo <br/> 	hanno tante sfaccettature. <br/> 	Se ti avvicini <br/> 	riflettono luoghi lontani <br/> 	che non immaginavi. <br/> 	E te ne innamori.</p>
<h3><a name="22"></a>Ricordo di una pentola dove c'era l'olio, ora piena di acqua - 20/07/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Una vita fatta di gocce <br/> 	di latte, di vino, di sangue <br/> 	mosaico di lagrime d'essenza <br/> 	costellazione di rimpianti, <br/> 	i ricordi girano <br/> 	ormai <br/> 	in orbite più distanti.</p>
