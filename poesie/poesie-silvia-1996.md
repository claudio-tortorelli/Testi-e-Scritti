<h2><a name="1"></a>Tra il tigri e l'Eufrate - 27/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Camminavamo, in un'area sfumata, <br/> tra il verde e il rosa. <br/> Parlavamo, di argomenti indefiniti, <br/> ma concreti. <br/> La tristezza e la gioia <br/> erano solo lo sfondo. <br/> In primo piano, finalmente, i nostri sguardi muti. <br/> Da qui 1000 volte, <br/> 1000 strade si diramano, <br/> da qui.</p>
<h2><a name="2"></a>Un cuore d'Acciaio - 27/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Quella volta <br/> mi chiese dell'orologio <br/> che portavo al braccio. <br/> "E' rotto? Perché non lo cambi?" <br/> "Non voglio cambiarlo, <br/> né tanto meno posso aggiustarlo." <br/> Ero bambino, quando mi avvolse il polso. <br/> Da allora i suoi tikkettii <br/> e i battiti del mio cuore si sono sincronizzati <br/> sempre più. <br/> Era con me nei giorni tristi e quelli felici <br/> nei giorni caldi e in quelli freddi <br/> nelle piccole-grandi imprese riuscite-fallite. <br/> Era con me anche quando ti conobbi <br/> e il mio cuore iniziò a pulsare così forte che <br/> lui non ce la fece <br/> o volle rimanere indietro, <br/> smettere di ricordarmi <br/> il trascorrere di un tempo splendido. <br/> Lei guardava avanti, lontano. <br/> Finalmente i suoi occhi si fissarono su di me <br/> e sorrisero.</p>
<p class="indent">
Ma la notte, alle volte, è breve, <br/> e il risveglio mi rubò il suo primo bacio.</p>
<h2><a name="3"></a>Lei e le stelle - 25/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Che spettacolo dalla bellezza e dal fascino viscerale! <br/> Lei non mancava di notarle, <br/> anche attraverso la luce  <br/> violenta delle città, <br/> la prigione estiva. <br/> Forse  <br/> anche lei viene da lassù, <br/> mi chiedevo, mentre con gli occhi <br/> salutava le sorelle. <br/> Mi prendeva la mano, <br/> e in un brivido  <br/> provavo tutta la sensazione  <br/> dei viaggi siderali. <br/> Ma non osavo rivelarglielo. <br/> Il silenzio parlava per noi  <br/> ma da dentro le dicevo <br/> non volare tra loro,  <br/> di illuminarmi più forte di questi anonimi  <br/> lampioni <br/> che l'uomo sfrutta per evitare di provare <br/> emozioni.</p>
<p class="indent">
Ma la notte, alle volte, è breve, <br/> e il risveglio mi rubò il suo primo bacio.</p>
<h2><a name="4"></a>Reazione metabolica istintiva - 29/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
E il velo si squarciò <br/> lasciando fuggire via <br/> l'intesa liquida  <br/> in cui pescavamo entrambi <br/> sogni. <br/> Quanto sono diversi i silenzi! <br/> Ora respiravamo quello dell'imbarazzo, <br/> mentre gli occhi si abbassavano. <br/> Poi la vidi spaziare lontano <br/> tornare a illuminarsi, <br/> sorridere salutare, <br/> ignorarmi dimenticarmi. <br/> Anche qui, qualche volta <br/> l'imprevisto scontato, non inevitabile, <br/> ci attende, <br/> e mi lascia da solo <br/> insieme a me stesso, <br/> a fare a pugni <br/> con me stesso. <br/> Ma questo è il mio territorio, <br/> quindi qui posso tornare indietro <br/> raggiungerli, <br/> e tra la comprensiva incomprensione generale <br/> sconfiggere l'imprevisto demolitore. <br/> Finito. <br/> Ho le mani insanguinate, <br/> e lei mi lascia,  <br/> per tornare nel suo mondo.</p>
<p class="indent">
La notte è breve, alle volte, <br/> e il risveglio mi rubò il suo primo bacio.</p>
<h2><a name="5"></a>Più di la che di qua - 22/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Commant ja peux fermer  <br/> Ce monde? <br/> Pochi granelli di sabbia <br/> Bastano <br/> a mandare all'aria i miei meccanismi. <br/> Et alors, <br/> alle volte  <br/> questa vita <br/> è un sogno <br/> altre <br/> un sogno <br/> diventa vita. <br/> E non so come svegliarmi <br/> o addormentarmi volontariamente, <br/> Chiamo in aiuto <br/> apro il mio cuore  <br/> a fantasmi <br/> e non mi accorgo che sono <br/> intorno a me, <br/> purtroppo <br/> sotto forma ben più consistente. <br/> E tutta questa post-pseudo-felicità <br/> è dovuta solamente <br/> ad un amore?</p>
<h2><a name="6"></a>Nascita, vita e morte di un'allucinazione estiva - 21/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Colline, gialle verdi <br/> colline si perdono <br/> sotto il cielo non più nero <br/> né rosa-arancione, <br/> ma già azzurro, <br/> celeste polveroso è l'orizzonte. <br/> Esco di casa, <br/> nell'immobilità, <br/> suona una campana, <br/> e mentre passo dall'ombra <br/> alla luce, <br/> sento il mio collo, le mie spalle <br/> avvampare, <br/> una cicala se ne accorge <br/> e zittisce. <br/> Le strade del paese <br/> convogliano l'odore, <br/> di grano mietuto, <br/> e mi riportano <br/> all'afa dei gialli campi <br/> che scorgo, tra due muri <br/> non lontani. <br/> Sudano le mani, suda la mente. <br/> Una lucertola se ne accorge e fugge. <br/> Come vedo <br/> neanche sotto questo vestito <br/> di piombo <br/> c'è sempre immobilità, <br/> né per l'anima, <br/> c'è ragionevole calma. <br/> A stento passa <br/> per l'angusta entrata <br/> di una chiesina. <br/> Luce-ombra. Ombra-luce. <br/> Caldo-fresco. Fresco-caldo. <br/> Il trauma si ripete <br/> anche all'esterno di me. <br/> Mentre una vecchina <br/> orlata di nero <br/> sgrana un rosario. <br/> Intanto qualcuno sciala, <br/> dalle persiane chiuse, <br/> una radio svolge mestamente il suo lavoro. <br/> Alla fine mentre continuo a vagare freneticamente <br/> me ne convinco, <br/> del mio errore. <br/> Neanche qui, neanche ora. <br/> L'immobilità, la pace, la calma <br/> non esistono neanche se imposte, <br/> da tutte le regole <br/> logiche, razionali, <br/> di un gioco stagionale. <br/> Rientro a casa <br/> con l'anima di nuovo ferita <br/> in una delle sue corse, <br/> seguito dalle grasse risate <br/> della lucertola, <br/> della cicala.</p>
<h2><a name="7"></a>Eccallà (Epilogo triste) - 17/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Stasera, l'ho cercata, <br/> e l'ho addirittura trovata. <br/> Mi è passata accanto, <br/> i nostri sguardi si sono incrociati, <br/> due parole sono emerse. <br/> Mi è passata accanto, <br/> ma come il vento, <br/> sentivo che non la potevo guardare, <br/> fermare, <br/> che altrove aveva la sua meta (o metà). <br/> Non so quanto si notava, <br/> ma stasera sorridevo senza motivo <br/> forse a lei <br/> forse a me stesso, <br/> l'incapace. <br/> E' proprio vero: <br/> quando presi la penna <br/> per scrivere questa storia, <br/> avevo in mente un altro finale. <br/> Se la modifica è stata giusta, opportuna o scontata, <br/> se la colpa è di qualcuno, <br/> non riesco a deciderlo. <br/> Mi sento muto, <br/> il mio amore è muto. <br/> Mi piace ancora pensare a quando era bambina, <br/> e che in questo momento sta dividendo con me i nostri 18 anni, <br/> ma questo rimane chiuso in uno scrigno <br/> affondato in fondo al mare <br/> dell'impotenza. <br/> Perché non posso avere ciò? <br/> E poi, questo epilogo, <br/> l'avevo anche già scritto, <br/> condannato. <br/> E continuare a sbagliare avvilisce. <br/> E sbaglierò ancora? Probabile..... <br/> No! Non si ripeterà, <br/> perché non vedo come possa ripetersi. <br/> Trovare l'amore, l'amicizia, la giovinezza, <br/> la felicità delle piccole cose, la timida bellezza, <br/> in un essere, solamente. <br/> Avrei anche avuto possibilità <br/> di variare la storia, <br/> che la trama lo consentiva <br/> (e questo mi fa male) <br/> ma non l'ho fatto, <br/> e ora il libro è alle stampe. <br/> Sono solo, <br/> pieno di amore irrisolto, incontrollato, scoraggiato, inconcluso, <br/> che non so come e se è ancora il caso di esprimere, <br/> che forse nessuno vuole, <br/> o che non vale nulla. <br/> Forse stasera, <br/> la lenta agonia <br/> si è conclusa. <br/> Qualcosa è morto, ma io, <br/> già con il lutto al braccio, <br/> ancora ironizzo. <br/> Solo alla morte non c'è rimedio, <br/> si dice. <br/> Non mi importa, <br/> quello che si dice. <br/> Anche se tutto a questo punto, <br/> per chiunque sarebbe, <br/> privo di senso, di scopo, <br/> io mi sforzo di dire, <br/> (sono un bischero) <br/> che ti voglio bene, <br/> ancora, <br/> e non voglio, <br/> per masochismo, <br/> dimenticare. <br/> Amen.</p>
<h2><a name="8"></a>Ombre cinesi - 15/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ovunque, con lei starei, bene, <br/> tanto che, <br/> comincio a domandarmi, <br/> chi è. <br/> Comincio ad avere paura. <br/> Che lei sia il sole, per me. <br/> Che io sia come un albero, per lei. <br/> Che la sagoma sul terreno <br/> delle mie foglie, <br/> colpite dalla luce, <br/> svanisca, si dissolva, sia assorbita, <br/> al passaggio della nuvola. <br/> Che l'ombra <br/> diventi un bene troppo comune.</p>
<h2><a name="9"></a>La vigilia di domani - 13/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Domani parto <br/> lascio questi pensieri <br/> a metà, <br/> conclusi. <br/> Il pianoforte <br/> commenterà i miei passi <br/> con note alte <br/> veloci <br/> le scale seguiranno le mie scale <br/> in salita <br/> in discesa. <br/> Gioie e dolori <br/> rimangono magnificamente sospesi <br/> in un rassicurante oblio. <br/> Trovo la luna <br/> in un angolo di cielo più nero. <br/> Colgo insieme, sotto le stelle, <br/> la paura e la speranza, <br/> che questo viaggio, <br/> sia concluso.</p>
<h2><a name="10"></a>Come le Ombre della Sera - 12/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ricordo un cipresso. <br/> Ricordo la sua esile, toscana forma. <br/> Ricordo la sua ombra, <br/> di cui era tremendamente avaro, <br/> ma che poi un po' concedeva <br/> negli afosi pomeriggi d'estate. <br/> Ricordo la strada, <br/> bianca, polvere mescolata a sassi, <br/> che gli passava accanto <br/> e poi piegava a destra dietro la collina, <br/> verso chissà dove. <br/> Ricordo le sue litigate <br/> con le ginestre fiorite <br/> che tanto lo disturbavano. <br/> Ricordo la fretta con cui <br/> pedalando lo lasciavo. <br/> Ma immagino, <br/> ora che forse non c'è più <br/> quanto quel paesaggio ha perso, <br/> senza il suo totem, <br/> che indicando il cielo <br/> ha posto un limite, <br/> ha dato un riferimento <br/> alla gente di qui, <br/> da sempre.</p>
<h2><a name="11"></a>Pensiero - 11/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Le idee di un uomo hanno successo <br/> quando chi le ascolta <br/> le prende in considerazione, <br/> le capisce, <br/> le accetta o le rifiuta, <br/> ma intanto ci si sofferma.</p>
<p class="indent">
Lo stesso vento, <br/> alle volte, <br/> carezza i campi di grano, piega le spighe e passa; <br/> poi, in mare aperto incontra una barca e gonfia la vela.</p>
<h2><a name="12"></a>Un bambino - 11/07/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Un bambino cammina in un sentiero, <br/> con tanti palloncini, <br/> legati ad esili fili, <br/> che stringe nel pugno. <br/> Il bambino cammina <br/> guardando cosa ha intorno, <br/> ma di tanto in tanto <br/> un palloncino si perde, <br/> si buca con uno scoppio assordante, <br/> improvviso e immediato, <br/> o vola via, <br/> piano piano, <br/> sale, finché sovrapposta al sole <br/> la sua sagoma si perde. <br/> Dapprima quel bambino piange, <br/> ma chi gli ridarà il palloncino? <br/> E poi ne ha ancora tanti! <br/> In seguito, <br/> quando la strada alle sue spalle si allunga, <br/> l'uomo, non si dispera più, <br/> stringe il suo pugno <br/> ma distende la fronte <br/> quando ancora uno <br/> dei pochi palloncini rimasti, <br/> se ne va.</p>
<p class="indent">
La vita è troppo bella <br/> e il sentiero, visto dall'alto, troppo breve, <br/> per versare più di una lacrima <br/> su ogni sogno volato.</p>

<h2><a name="14"></a>4 Luglio 1996 - 04/07/1996</h2>

Manca una settimana, <br/> lunga, triste settimana, <br/> alla Pasqua del Signore. <br/> E' la Domenica delle Palme, <br/> la vita contro la morte, <br/> la morte contro la gioia, <br/> la gioia contro lo sconforto, <br/> lo sconforto contro il sereno, <br/> il sereno contro le nuvole, <br/> le nuvole contro la vita.... <br/> Con un ramo d'olivo in mano, <br/> nella giornata di primavera, <br/> l'uomo si chiede perché il filo della vita <br/> e quello della morte <br/> sono così abilmente annodati. <br/> Come mai nessuno li scioglie. <br/> Il chiaro si mescola allo scuro, <br/> rendendo ambiguamente semplice e complicata <br/> la stessa cosa. <br/> Aspetta uomo. <br/> Aspetta, e se non avrai intelligenza, <br/> abbi almeno pazienza. <br/> Pazienza di aspettare Domenica.</p>
<h2><a name="55"></a>Eclissi - 31/03/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Guardo il sole, il sole m'acceca. <br/> Chiudo gli occhi e ritrovo il buio, il mio buio. <br/> Ma quando li riapro, non trovo la luce. <br/> Vedo solo quello che non potevo vedere, <br/> sotto uno dei suoi aspetti. <br/> Posso osservare, per qualche breve istante, <br/> anche ciò che non comprendo. <br/> Ma il tempo per capire è breve, troppo breve.</p>
<h2><a name="56"></a>Discoteca - 31/03/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Le luci intermittenti, come la coscienza. <br/> La musica ossessiva come la malinconia. <br/> Il volume alto come lo smarrimento. <br/> Il movimento intenso, frenetico, scomposto,  <br/> come l'emozione. <br/> I ragazzi abitano questi Sabato notte, <br/> li vivono e finiscono per portarli sempre con se. <br/> I ragazzi uniscono la loro solitudine, proprio dove i loro gridi <br/> si perdono. <br/> C'è fascino e spavento in chi li osserva. <br/> Io non posso criticarvi, né aiutarvi, né divertirmi con voi. <br/> Lasciate solo che i miei sentimenti non risuonino da un altoparlante, <br/> ma dal battito ritmico del cuore.</p>
<h2><a name="57"></a>Point of no return - 21/03/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
L'inutilità delle mie azioni  <br/> trova spesso conferma nello squallore del monotono, <br/> del nuvolo-piovoso, <br/> della comunicabilità superficiale. <br/> Come uscire dall'ordinario <br/> senza diventare straordinario? <br/> Quale droga, quale alcol, quanto fanatismo o incoscienza <br/> servono? <br/> L'onda corre fino alla spiaggia, urla e s'infrange, <br/> ma ritorna silenziosa al mare. <br/> L'aquilone vola sorretto da un vento quasi eterno, <br/> che presto finisce, <br/> e l'aquilone tocca terra. <br/> Cosa trattiene l'uomo nella sua anonima quotidianità, <br/> se non la paura di non poterci tornare, <br/> di non poter essere più quello di prima? <br/> Anch'io provo lo stesso terrore, <br/> la stessa sensazione anfibia, <br/> lo stesso remoto e prudente freno. <br/> Eppure, là nell'universo, <br/> buchi neri, invitano ad un viaggio senza ritorno, <br/> che pochi hanno compiuto.</p>
<h2><a name="58"></a>Insoddisfazione - 21/03/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Scorre lenta, una pasta dolciastra, tiepida. <br/> Scorre e goccia giù per il collo, ti avvolge, ti appiccica. <br/> L'anima si inquieta, ma salde catene le impediscono di fuggire. <br/> Non c'è modo di liberarsi di quell'irritante prurito, <br/> almeno che tu non agisca, <br/> non provi a dare un senso al tuo vivere, <br/> ma generalmente, per oggi, è troppo tardi. <br/> Allora ti accorgi che nei suoi occhi, se lei fosse qui, <br/> troveresti la fonte per lavarti.</p>
<h2><a name="59"></a>La sfera d'acciaio - 15/03/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Prima vivevo nella mia sfera di acciaio da solo. <br/> La mia immagine deformata,  <br/> guardava in tutte le direzioni, <br/> ma trovava sempre se stessa, <br/> in un ossessivo, monotono, <br/> triste ripetersi. <br/> Poi, da quando la tua figura è apparsa, <br/> riflessa sulla superficie lucida, <br/> guardo a nord, a sud, a est, a ovest, <br/> e prima di rincontrarmi, <br/> trovo sempre i tuoi occhi. <br/> E non serve ruotarla, <br/> la sfera non ha un senso. <br/> Non serve schiacciarla, <br/> perché continuerà a riflettere. <br/> Temo soltanto che manchi la luce.</p>
<h2><a name="60"></a>Anime danzanti nella nebbia - 11/03/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Se i corpi materiali sono così labili e suscettibili al tempo, <br/> quelle che ho di fronte non sono altro che anime, <br/> mosse da una volontà più o meno forte, <br/> da una coscienza più o meno profonda, <br/> da una molla più o meno carica. <br/> La loro gioia, il loro pianto, la loro paura, la loro solitudine <br/> sono espresse solo in modo approssimativo <br/> dai loro volti, <br/> ma io non chiedo di svelare i sentimenti  <br/> che si celano dietro i loro occhi, <br/> perché neanche i miei sono condivisi. <br/> Ammiro e derido con tenerezza <br/> la loro voglia di eternità.</p>
<h2><a name="61"></a>Voli di pensiero - 05/03/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Da ogni viaggio torno cambiato. <br/> La mia mente vola sopra i boschi, <br/> i fiumi, le radure e gli stagni, <br/> le miserie e le assurdità, <br/> mentre il mio corpo rimane qua, <br/> disteso, la marionetta della mia volontà. <br/> I miei occhi vedono, sorridono, e disperano, <br/> cose non viste,  <br/> fino a quando tornano ad aprirsi, o a chiudersi. <br/> Il mio viaggio mi ha fatto conoscere molte cose <br/> ma quando mi serviranno? <br/> Come tutti i viaggi, <br/> mi ha fatto venire invece nostalgia <br/> della vecchia, snobbata, criticata, <br/> casa.</p>
<h2><a name="62"></a>L'Acqua Santa - 29/02/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Din, Don... <br/> Risuona il passo zoppo <br/> nell'andito oscuro <br/> di una polvere sacra. <br/> C'accoglie una vecchia, <br/> il fazzoletto in testa, <br/> la pannuccia scucita. <br/> Entra il prete ed io lo seguo.</p>
<p class="indent">
Come è difficile, scomodo, triste, <br/> notare che ogni casa ha una facciata, <br/> che volutamente, orgogliosamente, <br/> nasconde l'apatia al tempo e alla vita.</p>
<p class="indent">
Fuori, il primo sole,  <br/> ancora troppo invernale, <br/> ma già causa di stolti miraggi, <br/> irradia la viuzza, <br/> dove un bambino col pallone <br/> calcia ritmicamente.</p>
<p class="indent">
Dentro, quel sole, quel suono, <br/> trapelano appena, <br/> animando i vecchi soprammobili, le fotografie, <br/> che la vecchia ora, <br/> dopo aver ripulito, <br/> carezza.</p>
<p class="indent">
Scorrono veloci le preghiere, <br/> si troncano fatalmente le parole. <br/> Presto ci avviamo all'uscita, alla luce, all'aria, alla vita, <br/> con febbrile smania. <br/> La porta si richiude.</p>
<p class="indent">
La goccia di acquasanta, è evaporata, <br/> nel deserto della solitudine.</p>
<h2><a name="63"></a>La porta dei Sogni - 25/02/1996</h2>

<p class="indent">
Parla con me stasera la luna. <br/> Canta per me stanotte il vento. <br/> Suona per me ora la pioggia. <br/> Ma io penso ancora a te, <br/> che sei luna, vento, pioggia, <br/> per la banderuola del mio cuore.</p>
<p class="indent">
Lascia socchiusa, più tardi, <br/> la porta dei sogni. <br/> Affinché possa entrare, <br/> in punta di piedi, <br/> ad ascoltare con te, <br/> finalmente il silenzio.</p>
<h2><a name="64"></a>Desolazione - 20/02/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Camminavo, <br/> era mattina o forse tarda sera. <br/> Ero in città. <br/> Ero da solo. <br/> Era caldo, ma un caldo pesante, da brivido. <br/> La finestra si socchiudeva, mossa da una mano incorporea. <br/> L'automobile era parcheggiata vicino al bidone, <br/> da dove si sentiva camminare un animale. <br/> Una cabina del telefono coi vetri rotti <br/> era accostata ad un vecchio muro pieno di graffiti, <br/> scrostato per metà. <br/> Per terra le pozze riflettevano i nuvoloni plumbei, <br/> quasi pulsanti, portati via dal vento d'alta quota. <br/> Un odore di terra bagnata penetrava l'aria, appena mossa. <br/> Da un vecchio cartello pubblicitario, <br/> un viottolo invaso dalle cartine e dalle erbe, <br/> portava ad una casa più bassa, buia, di mattoni. <br/> Un neon illuminava il retro di una casa cantoniera, <br/> il binario morto, giù alla stazione,  <br/> ospitava il vecchio vagone merci invaso dall'edera, <br/> che finalmente si muove...si muove. <br/> Mi sveglio e mi accorgo che nulla è cambiato.</p>
<h2><a name="65"></a>La calma - 14/02/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Cercavo un luogo deserto, <br/> dove mi parlasse il vento, <br/> e quel luogo lo trovavo nella cantina del mio cuore. <br/> Lì spegnevo la luce e mi lasciavo trasportare... <br/> Una mano trasparente, delicata, spesso mi sollevava, <br/> e uscivamo dal buio di quella cantina. <br/> Salivamo in alto, per poi soffermarsi a vedere un cielo nuovo <br/> da terra invisibile. <br/> Sentivo l'aria plasmarsi intorno a me. <br/> Poi il buio. <br/> Invano continuavo a cercare quel luogo.</p>
<p class="indent">
Cerco un luogo deserto, <br/> dove mi parli il vento, <br/> e quel luogo l'ho trovato nei tuoi occhi, <br/> nei tuoi movimenti, <br/> nei tuoi sorrisi, <br/> nei tuoi pensieri. <br/> Ormai solo lì, spengo la luce e mi lascio trasportare...</p>
<h2><a name="66"></a>Quello che non si dice - 13/02/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ciò che non dici, lo pensi col cuore. <br/> Quello che non fai, lo vivi nel cuore. <br/> Lasciami stare, caldo sole, ho fallito. <br/> Forse domani potrò rimediare <br/> ma oggi ho fallito. <br/> Lasciami stare, fresco vento, devo pensare. <br/> Forse domani avrò dimenticato <br/> ma oggi devo pensare. <br/> Vivo il dolore quando c'è gioia altrui. <br/> Vivo la gioia nel mio dolore. <br/> Chi sono per fare ciò? <br/> Forse solo il più considerato fra gli esclusi.</p>
<h2><a name="67"></a>Solitudine urbana - 21/01/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Cammino solo, <br/> di ritorno. <br/> Cammino a testa bassa, <br/> di notte. <br/> I lampioni mi illuminano il cammino,  <br/> ma io non vedo. <br/> Saluto la gente che mi passa accanto, come fantasmi, <br/> che appaiono e scompaiono nella mia mente.</p>
<p class="indent">
Seguo la mia ombra, che mi segue, <br/> mi affianca, <br/> mi stacca, e poi divertita mi ricompare alle spalle. <br/> Sento odori, vedo colori, odo rumori sconosciuti <br/> che solo quando non sono io percepisco. <br/> I palazzi visti da lontano sembrano perfetti, <br/> ma alle loro basi, dove gli passi accanto, trovi solo <br/> il sordo squallore della città nuova.</p>
<p class="indent">
Ero uscito per non pensare, o per pensare ad altro. <br/> Invece, quando sei solo nella viuzza scura bordata di orti, <br/> o quando sei immerso nella folla anonima <br/> e violacea sotto la luce artificiale, ciò non ti riesce. <br/> Inizi a fremere, come di freddo, <br/> il petto ti brucia, vien voglia di piangere, <br/> la mente galleggia sotto vuoto e perdi la tua comune identità, <br/> per acquistarne una nuova, forse più vera. <br/> Una mascherata tristezza ti avvolge, e ti fa pensare. <br/> Una tristezza isolata, fuori luogo, ma comunque compiaciuta, <br/> ovattata. <br/> Vorrei essere a 1000 chilometri di distanza, a riflettere. <br/> Vorrei scappare, portarti con me dove per un attimo potessimo scambiarci i pensieri, confrontarli, e poi magari riderci. <br/> E non ti saluto, perché quello che vedi non sono io,  <br/> è un essere solo, che ha tutto, che vive in tutto, <br/> e nonostante questo <br/> non sa perché è triste.</p>
<p>
Chi è più triste di chi non si gode la felicità?</p>
<h2><a name="68"></a>La luce del mattino - 14/01/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Entra con gioiosa prepotenza, dalle persiane accostate. <br/> Riempie gli occhi, <br/> ma non li ferisce. <br/> E' la luce che porta i colori, è la luce che porta gli odori. <br/> Mi alzo, <br/> apro le imposte, <br/> e l'obsoleta diga delle mie palpebre traballa e cade. <br/> Quante volte quei colori sono entrati in questa stanza, <br/> nelle mie giornate di primavera <br/> a farmi sentire la freschezza dell'aria, del cielo, della vita.</p>
<p class="indent">
Quante volte mi hanno fatto sentire pronto, <br/> eccitato, sicuro, ottimista. <br/> Quante volte, quella luce, sempre così schietta, <br/> azzurra, chiara, amica, rassicurante,  <br/> ho fatto finta di non vederla, <br/> per paura,  <br/> chiudendo vanamente gli occhi.</p>
<h2><a name="69"></a>La notte - 08/01/1996</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La notte! <br/> Nero spettro, <br/> cali laddove non è ancora arrivata la mia coscienza. <br/> E sussurri, e mi nascondi sempre la metà. <br/> La notte è grigia per chi riposa. <br/> La notte è nera per chi, solo, <br/> tra quattro muri, guarda in sù ed è sveglio. <br/> Guardo la luce dei lampioni tra le persiane accostate, <br/> commovente ribellione dell'uomo al suo destino. <br/> Ascolto, l'orologio che ticchetta nervosamente, <br/> l'automobile con chissà chi dentro, <br/> il treno che passa di nascosto, <br/> l'uomo che parla da solo, <br/> ..... <br/> poi anche i rumori sfumano e solo il pensiero ti costringe <br/> alla penosa veglia. <br/> Come nel dolore, nella notte sei solo, malgrado tutti. <br/> E da solo piangi, perché ci sono cose che non puoi capire,  <br/> a cui non puoi rimediare, che non saprai affrontare. <br/> Qui, maledetta, mi lasci sempre nudo, con i miei fantasmi. <br/> Mi lasci sempre ferito, ma mai a morte, maledetta! <br/> E poi, con il sadismo tipico delle cose vere, <br/> ineluttabili, <br/> mi lasci, prostrato, ma quasi in debito con te.</p>
<p>
Mi affaccio alla finestra.</p>

Il mio respiro caldo sfuma in vapore, che si confonde nelle nebbie dorate della nuova aurora.</p>
