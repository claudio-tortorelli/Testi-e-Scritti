<h2><a name="1"></a>Quota 1 - Orvieto</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Piove dagli anabbaglianti <br/> al selciato lucido e gonfio <br/> a tratti <br/> di tentacoli occulti. <br/> Scende dai platani  <br/> nella sera <br/> il penetrante <br/> crisi di memoria <br/> sfuocato passato autunno <br/> acre di fumo e di corteccia <br/> odore. <br/> L'ultima funicolare torna <br/> antibiotica <br/> alla base <br/> e il piede salta <br/> l'acqua che incanalata scorre, <br/> e porta un abito vuoto <br/> alla caserma. <br/> La mia mente nel freddo e nel buio <br/> cerca casa. <br/> Orvieto di sera.</p>
<h2><a name="2"></a>Quota 2 - Orvieto</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Camerata da dove vieni <br/> e dove andrai. <br/> Siamo due granelli  <br/> caduti nello stesso istante <br/> nella clessidra del destino.</p>
<h2><a name="3"></a>Quota 3 - Orvieto</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Una luce blu <br/> e due occhi pesanti <br/> aperti nel cuore <br/> della notte sul nulla <br/> di quest'istante. <br/> Di guardia stiamo <br/> in silenzio <br/> alle brande vuote <br/> alle brande piene. <br/> Fuori scorre lenta la luna <br/> e non pensi più <br/> che quello che fai serve, <br/> non pensi più.</p>
<h2><a name="4"></a>Quota 4 - Orvieto</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Gore d'umidità <br/> s'aprono al mio passo <br/> Mosè ridotto <br/> in un bagno <br/> scelgo un luogo dove stare <br/> solo, <br/> ma trovo altri pensieri <br/> dietro le porte. <br/> Solo una chiave cerco <br/> per isolarmi <br/> devo chiudermi <br/> e lasciare morire quello là fuori <br/> robot danzante <br/> già formattato <br/> dalla realtà e dal futuro. <br/> Per sopravvivere <br/> devo uccidere e tagliarmi a metà.</p>
<h2><a name="5"></a>Quota 5 - Orvieto</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Stacco tenue <br/> dolce filo di luce <br/> dorata <br/> che filtra una foglia <br/> l'essenza d'autunno <br/> illumina gli alberi al tramonto <br/> che tramontano anch'essi <br/> nell'arancio-verde. <br/> Viti ed olivi <br/> campane ed erba <br/> gentile si muove <br/> al vento di caldarroste <br/> di funghi <br/> ribolle il terreno <br/> e la campagna tutta <br/> punta l'ombra dei cipressi <br/> sempre più radi <br/> dove l'occhio spazia <br/> ad est. <br/> E d'ambra s'ammanta la valle <br/> e una casa lontana, <br/> un cane abbaia. <br/> Le scogliere dell'antica urbe <br/> mi sovrastano <br/> mi spingono a voltarmi lontano <br/> finché c'è luce <br/> azzurro limpido. <br/> Così l'occhio copre le distanze <br/> un boschetto di lecci <br/> un sentiero e un pollaio <br/> l'autostrada e i colli dove sparisce <br/> la mente <br/> va oltre, <br/> più lontano fugge via <br/> plana oltre il rosa ondulato e nostalgico dell'orizzonte <br/> via da questi giorni. <br/> Vita persa... <br/> Mi fermo al limitare di un orto <br/> tra le rotondità smeraldo <br/> della necropoli, <br/> gli etruschi e le tante bocche che li hanno ingoiati <br/> in fila <br/> epiche li aspettano. <br/> La luce ancora illumina il tumulo <br/> tufo e muschio <br/> odore di sonno <br/> invincibile silenzio <br/> violato. <br/> Anche la pace di un tramonto finisce <br/> dolce  <br/> una foglia rotola dentro ai giacigli vuoti <br/> e disfatti. <br/> Cambia il colore dei pioppi in lontananza. <br/> E uno spiffero freddo gela <br/> la Sicurezza eterna. <br/> Mi scende nel colletto <br/> una lama si insinua  <br/> alzo gli occhi nel nero <br/> annuso l'inganno della sera <br/> e dell'inverno, <br/> tradimento <br/> giuda autunnale <br/> re Mida che tocchi il cibo <br/> di uomini poveri e senza denti. <br/> Perché si domanda <br/> perché nulla è quello che è <br/> perché la mia vita appare accidentata  <br/> solo quando una luce la illumina male, <br/> mi ferisco i piedi e mi divincolo dentro <br/> poi mi volto e non vedo più sassi... <br/> allucinazioni <br/> sconvolgono la verità <br/> allucinazioni <br/> mi impediscono di rassegnarmi <br/> allucinazioni <br/> mi legano al domani.</p>
<p class="indent">
La Natura o il destino <br/> non promettono  <br/> così non mantengono. <br/> Non illudono  <br/> offrono oceani di acqua sporca  <br/> per i sazi <br/> limpida per gli assetati. <br/> E solo la stessa acqua.</p>
<h2><a name="6"></a>Quota 6 - Arezzo</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Uno squarcio sul muro <br/> un occhio pesto <br/> umido <br/> squallore interminabile <br/> prigione soffocante dei sensi <br/> delle intuizioni del vivere comune, <br/> scorre un tempo prezioso <br/> lavato via <br/> per bagni e camerate vuote <br/> brande scarne e senza memoria <br/> volti vitrei e freddi <br/> si muovono come fumo nell'aria <br/> inerti e lontani. <br/> Qui si spreca l'inconservabile <br/> il mattino siamese alla sera <br/> il giorno è spento dalla monotonia <br/> azioni insensate <br/> e muore senza lasciar braci <br/> ma solo polvere e cenere <br/> da soffiar via.</p>
<h2><a name="7"></a>Quota 7 - Arezzo</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Mia scelta o mio destino <br/> l'essere anonimo <br/> senza l'infamia, non meritevole di lode <br/> mai compiutamente vincente e neppure <br/> costretto da cause remote <br/> alla dolorosa, purificatrice, eletta <br/> sconfitta, <br/> vanto e gloria di chi si deve rialzare <br/> dignità <br/> di chi lotta e ha lottato <br/> domani. <br/> Il folle egoismo mi stringe e mi frusta <br/> verso scogliere d'invidia <br/> chi ride e piange più di me. <br/> Eterno rimpianto <br/> di essere l'incompleto <br/> di essere potuto essere <br/> e invece sono solamente <br/> io. <br/> Pazzo ancora fremo all'idea <br/> di rimanere appiccicato all'unto della mia condizione <br/> quando c'è chi, lontano <br/> patisce privazioni ed esperienze <br/> a me ormai negate.   <br/> Si fregia il basco <br/> di intima gloriosa tradizione. <br/> Incontentabile insopportabile resto a luci spente <br/> in una camerata vuota <br/> niente da vivere, nulla da ricordare <br/> da fare <br/> casco nell'apatia atrofizzante <br/> mentre le tante scuse con cui si cannoneggiano <br/> volontà e destino <br/> precipitano incrinandosi ai miei piedi. <br/> Non ho bocca che abbia voglia di lamentarsi <br/> né orecchie disposte a prestarle attenzione. <br/> Rimane tutto dentro me <br/> raccolto in un istante <br/> perché nessuno giustifica <br/> un'altra malsana allucinazione <br/> tormento offensivo per il comune buon senso. <br/> In questa guerra i morti si seppelliscono vivi. <br/> Tanti militi ignoti <br/> i miei giorni.</p>
<h2><a name="8"></a>Quota 8 - Arezzo - A Marilda - 07/12/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Immagina. <br/> Cammini in una selva <br/> tempo di nubi e fumi di ombre <br/> guazzando ungono i rami <br/> le foglie  <br/> tappeto agonizzante scivolano l'una sull'altra <br/> si uniscono anonimamente belle <br/> solo nell'insieme. <br/> Ha smesso di piovere, <br/> esposta ai venti che limano i monti <br/> sei avvolta. <br/> Con cura eviti l'acqua <br/> le pozze terracee e il fango ghiacciato, <br/> scansi quello che ti fa stare male <br/> conservi un po' di calore che ti conforta. <br/> Un passo sulla destra, avanti, sopra il sasso <br/> a sinistra, ancora avanti, <br/> Avanti, il più lontano, il meglio possibile <br/> asciutti, puliti, sani. <br/> Un percorso. <br/> Ma d'improvviso un colpo  <br/> sulla spalla una goccia <br/> due <br/> il gelo sulle membra <br/> il caso ha colpito. <br/> La gola gonfia ostacola l'aria <br/> una febbre corrode i tuoi occhi, <br/> rallenti il passo. <br/> A cosa è valso studiare il percorso <br/> calibrare passo e respiro? <br/> Il freddo temuto  <br/> piovuto dall'alto <br/> ti ha fermato <br/> ti aspettava da prima <br/> della tua partenza, della tua nascita <br/> su un ramo, <br/> per portare freddo e debolezza, <br/> per renderti impotente e non hai colpa <br/> e nessuno è responsabile. <br/> Però adesso non cammini più <br/> gli occhi lacrimano <br/> e il vento continua a scorrenti addosso. <br/> Più piccola ogni giorno <br/> vedi chi ha tanto calore <br/> sorpassarti <br/> indenne tra i rami ciechi <br/> che ti hanno accecato. <br/> Fredda e languida si abbatte la pioggia <br/> su tutti quanti <br/> ma tu sei ancora nel bosco <br/> senza colpa. <br/> E la dea del bosco <br/> adesso sei tu <br/> colei che conosce la fame e il freddo <br/> quella che ha perso la voglia di crescere <br/> e cresce di nascosto <br/> tra rami e fronde, senza più tetto <br/> si aggira e scruta dal bosco <br/> con grande tristezza, immensa rassegnazione, suprema dignità <br/> chi è già al di fuori. <br/> Intanto sull'albero che ti ha colpita <br/> un fulmine di rabbia  <br/> perché non ha merito e non ha colpa <br/> nessuno.  <br/> Ma tu sei già lontana. <br/> Forse un giorno anche lei uscirà.</p>
<h2><a name="9"></a>Quota 9 - Arezzo - 07/12/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Tanti fascicoli <br/> rilegati con cura <br/> tanti capitoli e fogli volanti <br/> uno sopra l'altro  <br/> la polvere copre lentamente anche l'ultimo <br/> ancora da chiudere. <br/> Una collezione mai completa <br/> un ordine provvisorio <br/> senza cataloghi <br/> né indici <br/> da cui pescare con sicurezza. <br/> Resiste al tempo la copertina <br/> si logorano le pagine interne. <br/> Lo spazio è alle volte riempito <br/> dal vuoto. <br/> Quel vuoto è parte di me.</p>
<h2><a name="10"></a>Quota 10 - Arezzo - 09/01/1998</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La sera dopo cena <br/> i miei passi cadenzati <br/> la nebbia e l'umidità <br/> un parco dimenticato <br/> di notte <br/> un sottopassaggio  <br/> segreto e riservato <br/> nelle sue luci sporche <br/> a pochi costretti. <br/> Si pensa anche per non pensare. <br/> Ma si cade <br/> nel gioco del se. <br/> Se fossi <br/> se fosse accaduto <br/> sarei... <br/> ... solo una goccia <br/> caduta in una scatola <br/> colma di sabbia, <br/> s'insinua tra mille granelli, <br/> trova infinite strade <br/> obbligate forse <br/> o bivi oppure <br/> dove proseguire <br/> la sua corsa verso il basso <br/> il fondo invisibile. <br/> Quante combinazioni! <br/> La volontà per i deboli <br/> e il destino per i forti <br/> portano all'inesorabile <br/> evaporazione. <br/> E si risale...</p>
<h2><a name="11"></a>Quota 11 - Orvieto</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La bocca ha labbra secche <br/> da troppo tempo non vi passa l'umida passione <br/> di una parola <br/> e la lingua si dibatte <br/> cieca d'angoscia <br/> come l'anguilla nel catino.</p>
<p class="indent">
Sto viaggiando e non so se tornerò <br/> se veramente sono partito <br/> con chi tornerò. <br/> Ma ogni momento la sabbia scorre <br/> sul mio viso <br/> allargando le crepe <br/> erodendo <br/> smagliando il tessuto celeste <br/> che avevo addosso.</p>
<h2><a name="12"></a>Quota 12 - Orvieto</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Rame sull'albero <br/> e sulla strada le scaglie <br/> del medesimo metallo <br/> saltano turbinando al vento <br/> del mio lesto passaggio <br/> che non le coglie. <br/> Il cielo è d'argento <br/> e di piombo a tratti <br/> e si appoggia pesante <br/> sui colli invernali <br/> orlanti la vallata aperta <br/> bronzea di terra. <br/> Piove mercurio sul parabrezza <br/> e l'erba e il grano ne assaggiano <br/> ne muoiono gli sterpi e le macchie. <br/> Dal ponte di pietra <br/> sul fiume d'acciaio <br/> scorgo una barca di legno. <br/> Attende la piena.</p>
<h2><a name="13"></a>Quota 13 - Arezzo</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Stanze vuote <br/> entro ed esco <br/> passivo. <br/> Gocce rugginose <br/> macchiano <br/> e la polvere <br/> ricopre i letti, le sedie, gli armadi metallici <br/> uno specchio che non riflette, <br/> io. <br/> Caos dello squallore <br/> un'arancia marcia <br/> un vetro rotto, <br/> mozziconi di sigaretta, <br/> io. <br/> Abbandono e fuga <br/> scempio di cuori <br/> monotona fotocopia bianca e nera, <br/> io. <br/> Io sono gli altri. <br/> Nulla di male, in quello che accade, <br/> nulla di duro, niente di tragico. <br/> Nulla. <br/> Soffia aria, entra luce <br/> anche qui dentro, <br/> ma non emozione e sogno. <br/> Provassi malinconia, sentissi l'amore, <br/> nascesse un dolore <br/> quale buona nuova sarebbe! <br/> Invece solo <br/> stanze vuote <br/> entro ed esco passivo.</p>
<p class="indent">
Si soffoca, si brucia senza clamore <br/> come candele ci si consuma <br/> sempre uguali a vista d'occhio <br/> sempre più piccoli. <br/> Sono Caino <br/> perché ho strozzato Abele. <br/> Ma dove fu odio, necessità, paura e gelosia, <br/> qui v'è soprattutto noia. <br/> Ho ucciso me stesso anche oggi <br/> e questa tenia ha bevuto un altro giorno, <br/> dei miei. <br/> Nessuna esistenza non si sbriciola <br/> ma qualche volta ci frana addosso un muro <br/> e ci accorgiamo del danno <br/> che non abbiamo tempo di restaurare, <br/> non si può. <br/> E la pioggia così dilava <br/> leviga le rovine, <br/> stacca infine ogni intonaco. <br/> Ed io mi sento veramente impotente di fronte a ciò. <br/> E divengo sempre più vuoto <br/> divengo cieco.</p>
<h2><a name="14"></a>Quota 14 - Arezzo - Ore 4,00 Garitta</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Due chiome si fronteggiano schematicamente <br/> nello stile proprio delle caserme <br/> alberi e muri <br/> dipingono il mondo al di qua. <br/> I fili elettrici volano <br/> sullo spazio di cielo rosso <br/> che scorgo dal vetro <br/> sporco di sonno <br/> della Garitta. <br/> E un neon bianco <br/> glaciale illumina una porzione di sogno <br/> sonnambolico. <br/> Chimere popolano la notte fredda <br/> di un febbraio morto <br/> sagome come ombre si proiettano per qualche momento <br/> contro le pareti. <br/> Ed uccelli <br/> urlano <br/> passando, colorandosi d'acciaio, e sparendo <br/> l'unico sinistro e lontano <br/> messaggio di vita <br/> imminente. <br/> E la normalità scompare <br/> il computo del tempo <br/> finalmente dilatato  <br/> per consentirmi di accorgermi <br/> della mia esistenza <br/> uccide per paradosso la mia anima giovane <br/> che invecchia e matura senza rendersene conto <br/> nutrendosi di un cibo denso <br/> insapore e incolore <br/> sopravvive ibernandosi. <br/> Il tempo troppo vissuto è perso?</p>
<h2><a name="15"></a>Quota 15 - Firenze</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Conosco la notte, e il freddo meccanico <br/> metallico del buio. <br/> Conosco l'addio, che ognuno impara <br/> a dimenticare <br/> alla fine del giorno. <br/> E l'addio che viene e ti prende. <br/> Come ti prendono i treni <br/> che trasportano singhiozzando <br/> e strisciando lenti. <br/> I treni alla partenza, <br/> che ti lasciano vedere cosa fugge  <br/> allo sguardo stanco di fuggire, <br/> amando ogni dettaglio che si perde <br/> non li puoi fermare. <br/> E corrono nel sentiero di nessuno <br/> non lasciano tracce <br/> non ti lasciano tempo. <br/> Si fermano quando sanno <br/> che non vuoi più scendere. <br/> I treni ti prendono.</p>
<h2><a name="16"></a>Quota 16 - Firenze - 13/08/1998</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Fresco risveglio <br/> Rinascere è bello <br/> Ogni giorno. <br/> Nel sonno oppresso dalla calda cappa <br/> Non sarò ormai più  <br/> Come non tornerò. <br/> Qualcuno si è perso nel limbo di questa bolla <br/> Chiusa ed esplosa. <br/> La grata filtra il giallo e l'azzurro <br/> I colori che immaginavo  <br/> Di notte <br/> Al freddo. <br/> Sono qui da solo <br/> Per un giorno qualunque. <br/> Il cortile interno <br/> È schivo agli sguardi e ai ricordi <br/> E con me rende le armi <br/> All'Alba.</p>
<h2><a name="17"></a>Quota 17 - Congedo - 13/08/1998</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Alle spalle ho il pianeta <br/> Il suo lento ruotare <br/> E' solo ricordo. <br/> Sulle spalle  <br/> La stella che brucia <br/> Una dopo l'altra <br/> Le emozioni e la frenesia. <br/> Il viaggio è iniziato <br/> E già sta finendo, <br/> ma nel deserto non si distinguono strade. <br/> Attendo <br/> La mia astronave rovente è lenta <br/> Cresciuto in qualcosa <br/> atrofizzato <br/> Torno. <br/> Rovente dondola <br/> Rallenta e riprende <br/> Mi rapisce e mi libera <br/> Ogni tanto. <br/> La polvere neutralizza il verde <br/> E di acqua non se ne trova <br/> E la pietra è spazzata <br/> E il cielo è bianco di luce. <br/> In mare non si trovano vie. <br/> Così immaginavo il ritorno <br/> Quando ad un tratto <br/> Ero a casa.</p>
