<h2><a name="1"></a>Falsa confessione - 23/02/97</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Fugace parola <br/> spiata negli occhi <br/> si abbassano, si mostrano, con colpevole sfida <br/> rimane il sale se l'acqua s'asciuga <br/> ma calore non c'è. <br/> Poi si ascolta, si finge <br/> o si sente di capire <br/> la fretta di fuggire leggero. <br/> L'uomo ha colpa o si crede colpevole? <br/> Forse lo è  <br/> comunque quando si pone questo tormento <br/> nel cuore <br/> e non trova la luce perché pensa al buio <br/> e nel buio rimane <br/> perché Dio non è il sole <br/> ma è i tuoi occhi che ti permettono di vederlo <br/> qualche istante. <br/> Ci si acceca, ci si confessa, <br/> ci si acceca, si dimentica, ipocrisia. <br/> Scaramucce di gabbiani sul mare. <br/> Serve al Mare il pescatore <br/> che nutra con i suoi scarti gli uccelli? <br/> Se il pescatore non uscisse <br/> forse molti morirebbero <br/> ingiustamente magari <br/> ma tra loro, qualcuno <br/> continuerebbe a cercare <br/> nel mare.</p>
<h2><a name="2"></a>Chiodo - 18/02/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Chiamala angoscia se non sai come <br/> trasmettere questo senso di nausea <br/> verso una giornata che non t'appartiene <br/> come volevi, <br/> una mattina nel tradimento <br/> mi sono svegliato. <br/> Parlo per me, di certo <br/> credo che altri non si soffermino <br/> da soli a pensare <br/> a questo cielo <br/> troppo azzurro per camminarci da soli. <br/> Io fuori <br/> escluso da un giorno <br/> nel quale tutti i miei sogni  <br/> si sarebbero avverati, nel quale <br/> nulla sarebbe successo. <br/> Ma l'anima sta male quando è costretta <br/> e quando è costretta alla libertà, <br/> esposta alle intemperie del diverso e dell'inconsueto <br/> si torce e si compiace <br/> in un dolore effimero che mai non svanirà <br/> del tutto nel ricordo. <br/> Parlo da una mattina <br/> rivolto ad una sera <br/> che aspetterà un'altra mattina <br/> nella speranza, <br/> e le gioie sembrano la causa dei dolori, <br/> necessari per una gioia. <br/> Ma così poca gioia esiste, <br/> nonostante il sole, <br/> quando la vivo <br/> lontano da te, <br/> per causa della mia stupidità. <br/> </p>
<h2><a name="3"></a>Pulizie - 16/02/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Strani pomeriggi <br/> che venivano qualche volta <br/> dopo i compiti di scuola. <br/> Aprivo l'alta finestra <br/> correvo in terrazza <br/> e vedevo nell'aria azzurrina <br/> fiori inesistenti: <br/> rose, tulipani, mimose, papaveri <br/> dall'essenza ne desumevo il numero <br/> e questo mi ubriacava. <br/> Pazzo di gioia di esistere <br/> di giocare alla vita <br/> quella vita che pensavo immutabile <br/> o solo lontanamente. <br/> Scendevo le scale di corsa <br/> nell'odore di cencio <br/> nei pomelli lustrati <br/> leggevo l'inizio di un'altra stagione (come?). <br/> Consapevole ignorante di una felicità <br/> spontanea, che ho fatto bene a notare e ricordare. <br/> Di lì a poco un frate con l'acquasanta.  </p>
<h2><a name="4"></a>Carnevale - 11/02/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Da quale giornale <br/> da quale mano <br/> arrivano strane stelle <br/> e cerchietti. <br/> Simbolo del caos <br/> loro origine, loro fine <br/> giacciono attaccati al pavimento <br/> da una bibita <br/> calpestati. <br/> Solo un breve volo <br/> nel cuore c'è già <br/> un cielo più basso su una terra più alta. <br/> L'allegria di una vecchia mazurka <br/> evidente falsità, <br/> ormai.  </p>
<h2><a name="5"></a>Cerco ragazza di anni 18 - 27/02/97</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Preferibilmente non usata, <br/> che mi chieda lei appuntamenti (ore pasti). <br/> Che mi guardi dolcemente, <br/> che abbia idee proprie. <br/> Che sappia ascoltare le cazzate, <br/> che le sappia ripetere di nuove. <br/> Che mi regali una sua foto, <br/> che mi faccia aspettare sotto i portici. <br/> Che mi rimproveri, <br/> che possa aiutare. <br/> Che superi la superficialità delle prime valutazioni, <br/> che mi dia una terza possibilità. <br/> E la sera quando vado a letto <br/> non c'è <br/> nessuna che mi sta pensando (penso), <br/> che sto pensando nel domani. <br/> E il domani arriva lo stesso.  </p>
<h2><a name="6"></a>Pomeriggio post-scolastico - 01/02/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Giochi di luce e di silenzio <br/> soli rimbombano <br/> dove di solito non li senti. <br/> A casa sono già  <br/> i volti 1000 ne scorri <br/> e dieci ne ricordi, <br/> e la scuola vuota  <br/> una muta malinconia. <br/> Le classi, passano tra le dita <br/> le aule nella luce pomeridiana <br/> piangono la noia di essere <br/> ignorate, si vede. <br/> Su qualche lavagna, voci di gesso <br/> le ore, le risa, le paure, le frenesie, <br/> la vita voltata alla ricerca di un possibile inizio <br/> e volata perché la gabbia dell'ultima campanella <br/> si è aperta. <br/> Passare tra i banchi e sfiorare chi vi sederà  <br/> domani, quanti segni di quante gioventù <br/> tentano di arrivarvi. <br/> Le due cupole ciarlano col duomo di morti e vespri <br/> matrimoni e domeniche mattina, <br/> e sotto da un po' le ascolto per un po' ancora <br/> nel cortile e l'atrio interno <br/> dove gli archi saltano di colonna in colonna, <br/> i famigerati bagni, alcune aule abbandonate, <br/> la palestra, un vecchio sgabuzzino delle scope, <br/> e poi,  <br/> la mia ultima stanza. <br/> Cammino nel presente di un edificio <br/> cha ha ospitato tanti presenti di coscienza, <br/> duri, secchi, incantati, innamorati, timorosi, curiosi, affaticati, sorridenti. <br/> E il ricordo ora, <br/> ora che li conto, <br/> mi duole e mi pesa già.  </p>
<h2><a name="7"></a>Schizzi del mare in burrasca - 26/01/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Lacrime ti illuminano. <br/> Strizzata hai l'anima. <br/> Coraggioso il cuore <br/> che sa mostrarsi debole negli occhi. <br/> E i tuoi <br/> condensano quell'essenza di sentimento <br/> che ti solca <br/> adesso il viso. <br/> Schizzi del mare in burrasca.	  </p>
<h2><a name="8"></a>Coce e frigge la salciccia fresca - 15/01/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Che bello quando vedi un prato rasato <br/> dal sole del giorno, le foglie strette <br/> dal caldo che ha fatto, ma ormai <br/> per oggi non farà più. <br/> Quando senti l'odore di terra  <br/> asciutta sotto un cielo normale ma <br/> estivo e nell'estate si perde <br/> tra le rondini s'imbrunisce <br/> coi pipistrelli. <br/> Maglietta corta e fremito <br/> sulla pelle che si bagna d'aria, e nell'aria <br/> essenza di vita, di festa, e di domani, <br/> di temporale e amore, di fumo e <br/> parole dimenticate. <br/> Che bello quando questo lo vedi e  <br/> lo provi senza che esista <br/> percependolo da una nota diversa <br/> sapendo che non lo apprezzerai <br/> quando sarà un istante qualunque.  </p>
<h2><a name="9"></a>Banale - 13/01/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Sorge come la luna <br/> di notte <br/> amaro, malinconico  <br/> sorriso storto nel blu, <br/> un sentimento vago in un cuore bambino, <br/> che si contenta di vivere <br/> lontano la vita. <br/> Sorge come dubbio <br/> tramonta come certezza, <br/> dal cielo crollano le braccia <br/> alzate, tese nello spasimo, <br/> adesso finalmente domate. <br/> La morte. <br/> Non ti è estranea, non è lontana e nemmeno eccezionale! <br/> Capita, prendi e impara, vivi. <br/> Questo c'è chi lo sa, <br/> anche troppo, <br/> non piange e non ride, <br/> soffre e spera, solamente <br/> nel silenzio, <br/> quando un occhio si appanna <br/> si schiude altrove. <br/> Felice chi muore di morte innocente, <br/> che non sa cosa si è perso. <br/> Felice chi muore di morte ragionata, <br/> che sa cosa si perde. <br/> Forse... <br/> felice chi muore sapendo che si deve <br/> lasciare che Dio ci insegni <br/> i limiti in cui crescere.  </p>
<h2><a name="10"></a>Soffitto bianco - 06/01/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Cinema, <br/> una volta così ti chiamavo, <br/> con la vita in bocca, <br/> in una stanza aperta <br/> su un mondo socchiuso <br/> pensare nella penombra, <br/> ingannarsi nel caldo, <br/> fermarsi nel tempo, <br/> lungo tempo sfuocato, <br/> come i colori, <br/> proiettati dalle auto ignare <br/> del loro passaggio <br/> sulla mia mente, <br/> distesa sul letto, <br/> con una nonna di fianco, <br/> ridere, contare, <br/> imparare ad imparare da tutto. <br/> Soffitto, <br/> rimani stanotte <br/> da tanto ormai lo sei <br/> uno specchio semmai <br/> di un'anima stanca <br/> nei torridi pomeriggi <br/> di rimirarsi, volare un po' <br/> poi sdraiarsi ancora <br/> a faccia in su, <br/> stanca di sapere di non sapere nulla <br/> stanca di non avere più nessuno <br/> accanto a dare corpo alle ombre, <br/> ormai testardamente definite <br/> in contorni ripetitivi, <br/> stanca di perdere <br/> spina dopo spina, <br/> la fantasia di vivere la propria fantasia.  </p>
<h2><a name="11"></a>Chador - 06/01/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Veste arcaica <br/> di antiche genti <br/> taglia di netto <br/> un volto. <br/> Vecchio, fresco <br/> bello, brutto <br/> candido, tostato <br/> non saprei, <br/> non so soffermarmi se non sugli <br/> occhi, <br/> unico messaggio <br/> dell'essere dinnanzi. <br/> Democratico <br/> caldo <br/> eterno discorrere <br/> di uno sguardo <br/> senza età, <br/> occhi <br/> ancor più nudi <br/> su un volto coperto. <br/> Fascino del mistero della donna, <br/> di una rappresentativa porzione di bellezza, <br/> l'ho visto in  <br/> occhi <br/> davvero impegnati nell'esprimere <br/> l'anima <br/> davvero abili nel nasconderla. <br/> Anche nei tuoi l'ho trovato, <br/> libera corona <br/> del tuo sorriso.  </p>
<h2><a name="12"></a>Collana di perle di plastica - 4 monologhi a Capodanno - 01/01/97 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La neve. <br/> Corre, lontano la voce <br/> che la neve, maledetta stupida <br/> sia caduta, <br/> cambiando ovunque un po' <br/> i contorni, <br/> rallentando un tempo <br/> troppo rapido, frenetico <br/> turbine di uomini <br/> che non sanno osservare <br/> un bianco fiore di ghiaccio <br/> scivolare in basso <br/> lentamente, lentamente.... <br/>  </p>
<p class="indent">
Io non c'ero, e lei <br/> non aspetta, donna timida, <br/> il primo, ultimo appuntamento <br/> con una giovinezza  <br/> ormai prossima a squagliarsi. <br/> Ero invece dove il sole <br/> rende umida la terra <br/> lungo una strada di campo <br/> saltando di erba in erba <br/> per non cadere nel fango. <br/> Fresco vento, gita di Marzo, <br/> corre di ramo in ramo, <br/> portami notizie da quei monti <br/> invece che odori che si perdono nei ricordi. <br/> Se sapessi scolpire quello che provo <br/> anche le pietre penserebbero <br/> a come esprimere il sentimento. </p>
<p class="indent">
Ricordo la sera in un'Estate <br/> quando ti pensavo nell'afa di un sogno <br/> come una canzone, una serata romana <br/> tra i vecchi palazzi, giallilluminati <br/> dai lampioni, sotto un portico <br/> nel parco nel caldo, <br/> quando finita la scuola <br/> ti perdi, ti perdo <br/> nel seguirti molto a modo mio. <br/> E allora ricordavo l'inverno <br/> che ti conobbi veramente <br/> che iniziai a conoscermi <br/> gridando al vento freddo, <br/> confidando alle volte <br/> alla luna terza, e quella <br/> qualche volta sorrideva. <br/> Ricordavo le mattine nebbiose <br/> le castagne, la scuola, il grigio <br/> e l'azzurro-ghiaccio <br/> come di un film <br/> si ricordano le comparse. <br/> La professione di chi sogna <br/> sta nel non credere nella realtà presente <br/> nel deludersi nello sperare <br/> infinite volte.</p>
<p class="indent">
Un'altra epoca è svanita <br/> senza nascere per tutti gli altri <br/> che vivono il tempo  <br/> scondito dall'orologio <br/> non quello rappresentato dai sentimenti  <br/> nella risacca, <br/> dal battito lentapatico o vitalemotivo <br/> del cuore. <br/> Eppure sono gli altri a far sorgere in me <br/> un nuovo anno, a farne morire uno vecchio, <br/> a farmi desiderare di saper scrivere, <br/> suonare, cantare, dipingere, parlare, amare, scherzare, crescere, vivere, <br/> invecchiare e morire. <br/> Sono loro che mi ricorda un cielo. <br/> Ma sono solo io, che lo traduco in un duro fardello <br/> alle volte <br/> in un solletico, per l'anima. <br/> Ho vissuto i mitici 18 anni <br/> in cui ognuno costruisce <br/> la parte più soffice e velata <br/> pur nell'estrema durezza, <br/> della sua vita. <br/> La parte che si ricorda <br/> socchiudendo gli occhi <br/> e fissando lo sguardo oltre la parete. <br/> Oggi iniziano altri 18 anni.  </p>
<h2><a name="13"></a>Parenti lontani nel tempo - 22/12/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Domenica quasi Natale <br/> cielo grigio quasi bianco <br/> la casa silenziosa quasi vuota <br/> in chi la abita <br/> oggi. <br/> Ne usciamo io e mio babbo <br/> senza chiudere la porta <br/> e la pendola continua a battere <br/> un tempo più lungo del solito <br/> trattiene la mente su penose circostanze. <br/> Una volta fuori <br/> un colpo di tosse  <br/> espelle la morte <br/> ingoiata di traverso. <br/> In viaggio c'è tempo <br/> di ricordare <br/> una persona vestita della sacralità <br/> del tempo <br/> che non vedrò. <br/> Ricordare la sua abilità <br/> la sua giovinezza <br/> la sua vita e i suoi amori, <br/> le emozioni e i sentimenti, <br/> le esperienze, <br/> che io non potrò. <br/> Ricordare.  </p>
<h2><a name="14"></a>Troppo presto e troppo tardi - 22/12/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Nato da solo <br/> fuori di casa <br/> non so dove andare <br/> ma soprattutto <br/> non so immaginarmelo. <br/> C'è chi ha amato, <br/> si dice che c'è <br/> chi ha creduto in qualcosa. <br/> Ne invidio l'ingenuità <br/> la libertà di pensiero <br/> che io mi devo inventare <br/> che nessuno mi ha insegnato <br/> per paura forse, <br/> per disillusione credo. <br/> Nato nei colori giallo-arancio-rossi <br/> dell'autunno <br/> trovo solo ricordi <br/> di un'infanzia primi anni'80 <br/> fine millennio, fine secolo, fine epoca, <br/> inizio... <br/> poche emozioni di mente, sentimenti di cuore, <br/> sotto un tiepido sole, <br/> solo nipote di stelle lanciate <br/> troppo in alto per essere ritrovate. <br/> Troppo tardi per vederle partire, <br/> troppo presto per abbracciare  <br/> un eventuale ritorno.  </p>
<h2><a name="15"></a>A chi è e non - 08/12/96 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Dicocca da un piano <br/> la nota, l'arpeggio, l'armonia <br/> sintetica prima, analitica poi <br/> delle tue percezioni sensimentali. <br/> Lascia lo schema dal carattere imposto <br/> se vuoi, se sai, <br/> bacia la mano, ringrazia la mente, trova lo strumento, <br/> per ricordarti per sempre <br/> la tua soggettiva e, vedrai, universale <br/> emozione.  </p>
<h2><a name="16"></a>Una ragazza e un cuore di strada - 01/12/96 </h2>
