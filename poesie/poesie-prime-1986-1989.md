<h2><a name="1"></a>Autunno ritorna - Settembre 1989</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Autunno ritorna <br/> e con le sue piogge <br/> diventano umide  <br/> le antiche logge. <br/> Con le sue foglie gialle <br/> pin piano manda via l'estate <br/> e si rigonfiano i fiumi <br/> e le cascate. <br/> Autunno ritorna, <br/> con la sua maschera di malinconia, <br/> ma che in realtà <br/> è allegria <br/> che solo i poeti <br/> possono coglier. <br/> .....Autunno ritorna.</p>
<h2><a name="2"></a>Il campo fiorito</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Eri fangoso, nero, squallido. <br/> Ora le spighe <br/> sono il colore dell'oro. <br/> Il cielo azzurro <br/> accoglie il sole <br/> che ogni giorno <br/> si fa più onore. <br/> I fiori sono sbocciati, <br/> gli alberi resuscitati <br/> dalla morsa dell'inverno. <br/> Bel campo! <br/> Una volta eri nero, <br/> ora metti in mostra <br/> i tuoi colori... <br/> E fra i tanti fiori, <br/> una margherita <br/> che, come una perla, <br/> luccica intontita.</p>
<h2><a name="3"></a>Passeggiata</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Passeggiavo per i prati, sono contento, <br/> i fiori sono sbocciati, <br/> freschi vivaci di tutti i colori <br/> fan diventare la campagna un arcobaleno  <br/> che pare una favola. <br/> Gli alberi son come rinati <br/> con i boccioli appena spuntati <br/> e il sonno invernale è cosa lontana <br/> perché solo acqua limpida sgorga dalla fontana. <br/> Il cielo azzurro, leggero, infinito, <br/> pare di stare in paradiso, <br/> quando d'un tratto qualcosa si muove. <br/> Sono le rondini che finalmente, <br/> come cani fedeli sono tornate,  <br/> e anche loro ci dicon "forza, gioite!".</p>
<h2><a name="4"></a>Autunno</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Molte foglie si disperdono per l'aria <br/> e il sole non c'è. <br/> Cade la pioggerella <br/> e l'ombrello non la para. <br/> L'aria è umida <br/> e i tetti sono pieni di piccioni bagnati.</p>
<h2><a name="5"></a>Il gelo</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Il gelo filtrando con forza fra i panni <br/> t'immerge fra il freddo dell'inverno. <br/> Fra i candidi fiocchi bianchi  <br/> i tetti rossi fumano dolcemente, <br/> mentre i pochi fiorellini <br/> vengono coperti da un sottile manto gelato.</p>
<h2><a name="6"></a>La primavera è arrivata - Febbraio 1987 </h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
E' arrivata dolcemente <br/> senza mostrare alcuna turbolenza, <br/> come un fiore <br/> che sboccia allegramente. <br/> Nessun alito di vento tira: <br/> è arrivata la primavera <br/> accompagnata dal caldo sole. <br/> Con il canto degli uccellini <br/> e la gioia dei bambini, <br/> con gli alberi fioriti <br/> che di verde son vestiti <br/> è arrivata la Primavera.</p>
<h2><a name="7"></a>Preghiera - Gennaio 1987</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Questo giorno mi hai dato <br/> pieno di gioie e di sconforti. <br/> Ora la tua forza mi porti <br/> a vivere la mia vita.</p>
<h2><a name="8"></a>Nevicata - Dicembre 1986</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Dormivo, <br/> mentre un soffice manto bianco <br/> si posava sulle strade. <br/> Ora è mattina: <br/> i bambini giocano contenti <br/> senza rispettare quel tappeto bianco <br/> che dondolandosi si è posato lievemente nella notte <br/> ed ora è sterminato dai raggi del sole.</p>
<h2><a name="9"></a>La primavera - Aprile 1986</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Le rondini volano sopra le fronde degli alberi. <br/> I ruscelli sono una meraviglia davvero <br/> con i campi fioriti e l'azzurro del cielo. <br/> Evviva la Primavera, <br/> che è bella davvero! <br/> Le api si posano sul fiore felici e contente: <br/> Evviva la Primavera!</p>
<h2><a name="10"></a>La neve - Febbraio 1986</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La neve cade giù dal ciel, <br/> cade giù dal ciel un po' come un fior strano: <br/> la neve scende piano e si posa come una mano <br/> leggera sulla terra.</p>
<h2><a name="11"></a>Claudio e Giulia</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p>
(IL VOSTRO CARISSIMO NONNO LUIGI)</p>
<p class="indent">
Tu Claudio ormai grandicello <br/> sano di mente e bello <br/> è tuo dovere dare <br/> buoni consigli <br/> a tua sorellina Giulia, <br/> brava bambina intelligente <br/> e bella <br/> che io vedo che brilla <br/> come una stella. <br/> Tiella sempre a te vicino <br/> tiella per mano <br/> e dai a lei sempre <br/> buoni consigli <br/> durante la vostra vita <br/> che è più minorenne. <br/> Aiutala nella scuola <br/> e in altre cose buone <br/> della vita. <br/> Non lasciarla mai sola <br/> ma tiella per mano, <br/> anche quando sarete adulti <br/> e con questo compito <br/> così arduo e fraterno <br/> sarete soddisfatti e felici <br/> per tutta la vostra vita <br/> colma di gratitudini.</p>
<p>
Luigi Tortorelli</p>
