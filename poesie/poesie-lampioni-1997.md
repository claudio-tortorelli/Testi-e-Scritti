<h3><a name="2"></a>Terremoto - 27/09/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Il muro crolla <br/> 	la terra <br/> 	la terra <br/> 	non regge <br/> 	nulla. <br/> 	Siamo paglia. <br/> 	I luoghi le immagini, <br/> 	le vite i progetti, <br/> 	sono perduti. <br/> 	Delirio di onnipotenza <br/> 	pensare al futuro. <br/> 	Perdiamo anche il nostro  <br/> 	ieri, <br/> 	e io non servo <br/> 	non sono già <br/> 	più <br/> 	mai <br/> 	lo stesso.</p>
<p class="indent">
La terra, <br/> 	la terra <br/> 	ancora. <br/> 	Il cielo estraneo e insolente <br/> 	sereno rovina <br/> 	grave  <br/> 	sugli occhi. <br/> 	E polvere e lamenti <br/> 	di esseri inconsapevolmente animati <br/> 	da uno sputo di vita <br/> 	ora soli e disperati <br/> 	inanimati <br/> 	senza ieri né oggi <br/> 	ed un triste freddo <br/> 	domani.</p>
<p class="indent">
Inutile stringere i pugni <br/> 	sui propri ricordi, <br/> 	l'umano orgoglio stracciato, <br/> 	una furia anonima <br/> 	senza volto né causa <br/> 	segna le mani  <br/> 	di sangue. <br/> 	Inutile stringersi gli uni agli altri <br/> 	inutile, inutile <br/> 	umano e fatale. <br/> 	"Perché il caso e il destino" <br/> 	?! <br/> 	si leva un coro <br/> 	che sale a chi non sembra ascoltare <br/> 	qui le macerie <br/> 	già rispondono duramente <br/> 	troppo <br/> 	per accettare <br/> 	che il caso sia il Caos plasmato <br/> 	per l'uomo chiamato <br/> 	destino.</p>
<p class="indent">
Riassunto di una stupefatta <br/> 	desolazione. <br/> 	Piegati i ginocchi <br/> 	di noi deliranti. <br/> 	Siamo grano  <br/> 	di innumerevoli campi <br/> 	tra infiniti mulini.</p>
<h3><a name="3"></a>Discorsi sul tempo - 04/09/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Casco dai tuoi discorsi <br/> 	nel mio sentiero passato e futuro <br/> 	il presente mi sta vicino. <br/> 	Mi chiedi del sentimento <br/> 	della sua purezza <br/> 	della sua rarità. <br/> 	Dell'amore anche mi chiedi un'opinione. <br/> 	Io invece pendolo <br/> 	tra due convinzioni <br/> 	opposte. <br/> 	Conosci già la risposta <br/> 	o sei la mia dantesca Beatrice? <br/> 	Cammino di fianco ad un bivio. <br/> 	Sorrido e navigando tra parole di gomma <br/> 	svengo nella corrente.. <br/> 	Un ragazzo che una volta conoscevo <br/> 	in uno dei suoi infiniti istanti <br/> 	affiora dalle tue labbra, <br/> 	che lo hanno sfiorato. <br/> 	Perché non bercio <br/> 	come chi sta per essere diviso <br/> 	dalle cose più care, <br/> 	stanco comunque di palleggiare <br/> 	in verticale il suo macigno? <br/> 	Invece mi chiedo cosa sono. <br/> 	Ma la risposta è sempre più stupida della domanda. <br/> 	Uno che ha lasciato i bonus <br/> 	per arrivare al mostro di fine livello? <br/> 	Uno che si chiederà sempre <br/> 	se quello che dicono le canzoni d'amore è vero? <br/> 	Uno che rimpiangerà le eredità <br/> 	mai seminate,  <br/> 	i troppi sogni, la fantasia? <br/> 	Uno solo. Uno e basta. Uno e mezzo. <br/> 	Sono di già il mio futuro. <br/> 	E siamo già alla prima panchina, seduti. <br/> 	Mentre tu mi parli seccata di cose che non ho <br/> 	potuto vedere malgrado la mia fortuna, <br/> 	limitato nei sensi  <br/> 	e cresciuto nella mente. <br/> 	Gioia=accontentarsi+pazienza+fortuna <br/> 	non regge come mio alibi <br/> 	Io giudice avvocato e imputato <br/> 	cento volte al giorno <br/> 	in tribunale si dibatte su cose perse <br/> 	non riscosse. <br/> 	Un bacio passivo <br/> 	mi ci vorrebbe sta' grazia <br/> 	sto' colpo di grazia <br/> 	mentre il sole precipita <br/> 	insanguinato <br/> 	dietro la tua testa e il mio alterego futuro <br/> 	è scolpito, sbozzato e modificato <br/> 	senza tregua <br/> 	sempre più finito <br/> 	sempre più immutabile <br/> 	sempre più vicino.</p>
<h3><a name="4"></a>Anni '70 - 04/09/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Canzoni anni '70 <br/> 	scivolano la notte <br/> 	una piccola frequenza <br/> 	li capta <br/> 	locale e lontana. <br/> 	Disperazione e passione <br/> 	scivolano la notte <br/> 	su di un letto troppo giovane <br/> 	per l'estate <br/> 	commosso e estraneo. <br/> 	Amore, chitarre estati diverse <br/> 	esistite anche loro <br/> 	rigano un cuore <br/> 	che vorrebbe sanguinare <br/> 	ma non ha e non è <br/> 	tempo. <br/> 	Perché.</p>
<h3><a name="5"></a>Fantarealtà in Sol - 03/09/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Soffia sale <br/> 	su un nuovo crepuscolo, <br/> 	muore il giorno e l'estate <br/> 	si chiudono le persiane <br/> 	nuovamente la sera. <br/> 	Tra le mani una radio <br/> 	che canta un sogno in fm <br/> 	inglese <br/> 	batte un ritmo lontano <br/> 	dal clima di strada <br/> 	qua sotto. <br/> 	Una saracinesca. <br/> 	Come non pensarti <br/> 	me lo insegnò un muro <br/> 	ma su questo ti proietti <br/> 	già donna <br/> 	sbiadita. <br/> 	"E t'ho voluto bene  <br/> 	ma non te l'ho detto mai <br/> 	sono stato un bischero <br/> 	sarebbe forse stato diverso <br/> 	ma io sono così <br/> 	e te rispecchi un lato fresco e bello <br/> 	di me <br/> 	che non conosco ancora <br/> 	ti chiedo solo di non dimenticarmi <br/> 	io non ti dimenticherò <br/> 	domani saremo lontani <br/> 	ma continuerò a pensarti." <br/> 	Un klakson. <br/> 	E sono già fuori da quel presente ideale  <br/> 	ma triste <br/> 	eroicomica d'amore, giovane. <br/> 	Una soffiata di naso <br/> 	e un colpo di tosse <br/> 	e domani devo ricordarmi di <br/> 	"Ciao, com'è? <br/> 	Ti ricordi di quando eravamo <br/> 	di quando dicevi <br/> 	e lo sai che fine ha fatto <br/> 	io invece sempre al solito <br/> 	nulla è  cambiato e conosco ora il mio domani <br/> 	io ti volevo dire che,  <br/> 	inutile rattristarci se si vive per ridere <br/> 	delle tristezze <br/> 	fatti viva qualche volta, <br/> 	ah, e saluta tuo marito."</p>
<h3><a name="6"></a>27/08/1997 - 27/08/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La schiuma <br/> 	che nelle notti idrofobe  <br/> 	trasforma anime licantrope <br/> 	per qualche istante <br/> 	in materia cosciente <br/> 	colando dalla luna  <br/> 	mi raggiunge sul prato. <br/> 	Contamina   <br/> 	questo crepuscolo <br/> 	appena terminato, <br/> 	già freddo. <br/> 	L'anno e la vita  <br/> 	ti accorgi finiscono <br/> 	in un momento <br/> 	in ogni momento.</p>
<p class="indent">
Due amici e una festa paesana <br/> 	aroma di lampione tra le siepi <br/> 	e qualche mazurka <br/> 	che sa sempre di finto vivere <br/> 	prossimo notturno futuro <br/> 	che questo luogo <br/> 	ormai solitario, <br/> 	affronterà.</p>
<p class="indent">
L'esistenza non offre nulla all'immobile. <br/> 	Sta a noi ruotare attorno al nulla <br/> 	tanti elettroni circondano un nucleo <br/> 	ma dove trovare l'energia per liberarsi? <br/> 	Il flash presente <br/> 	sgretola intanto <br/> 	un'altra <br/> 	fenice mai bruciata.</p>
<p class="indent">
E vivere finalmente soddisfatti <br/> 	e assieme consapevoli della propria soddisfazione <br/> 	tra tempo e tenebre, <br/> 	troppo bene e poco male <br/> 	abbandonati fiduciosamente <br/> 	tra una Scilla cieca e un Cariddi inesistente, <br/> 	ancora una volta sotto i piedi <br/> 	della speranza, dell'Amore.</p>
<p class="indent">
La schiuma di luna <br/> 	creata per questo <br/> 	piove languida sull'occhio <br/> 	e scivola eterea <br/> 	tra il cuore e la scapola <br/> 	accende di nuovo il Tormento.</p>
<h3><a name="7"></a>21/08/1997 - 21/08/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Cade cenere  <br/> 	dai lembi del mio vecchio abito <br/> 	che sto consumando, <br/> 	amore, <br/> 	al fuoco lento del nulla <br/> 	in cui mi è caduto, <br/> 	macchiato ormai. <br/> 	Giovani si è per sempre <br/> 	ma ci sono istanti <br/> 	che non lo sono mai stato. <br/> 	Partendo da te <br/> 	dalle tue coste, laghi e insenature <br/> 	già diverse <br/> 	dalla realtà, dal ricordo <br/> 	fino alla vita che vivevo <br/> 	adesso solo su un guanciale. <br/> 	Devo soffiare la cappa <br/> 	del nulla inerte  <br/> 	su me. <br/> 	Non già sull'estate apaticamente <br/> 	ibernante <br/> 	la mia illusione è nata. <br/> 	Sono morto una volta in più. <br/> 	Nel nulla  <br/> 	sciacqui di parole <br/> 	cozzano sulle sponde della mente <br/> 	corrodendo fino a bucarla. <br/> 	Un faro su un promontorio. <br/> 	La scia di un gabbiano <br/> 	blu <br/> 	azzurro <br/> 	grigio. <br/> 	All'orizzonte il nulla <br/> 	ed in mezzo una nave <br/> 	salpata all'alba. <br/> 	Controvento il sale <br/> 	morde la faccia <br/> 	quando un lampo   <br/> 	casuale <br/> 	turba la cosmica armonia <br/> 	del movimento marino.</p>
<h3><a name="8"></a>28/07/1997 - 28/07/1997	</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Generale da nord-est <br/> 	ti sento cavalcare nuvole pesanti <br/> 	un galoppo lontano e grave <br/> 	immenso solenne, viola e blu <br/> 	come la cenere che il vento non la porta <br/> 	quando è bagnata. <br/> 	Generale vieni e bagnami <br/> 	senza onori militari. <br/> 	Se sono un fallito <br/> 	ce ne saranno i motivi <br/> 	che comprendo con la ragione altrui. <br/> 	Ma fallire non basta <br/> 	a sentirsi fallito, <br/> 	bisogna riuscire a metà <br/> 	fermarsi sospesi, scivolando prossimi al traguardo <br/> 	naturalmente miraggio illusorio del proprio mito di completezza.</p>
<p class="indent">
Anonimo, ignavo, immobile, inetto. <br/> 	Credevo alle possibilità del mio corpo <br/> 	della mia mente e dei miei sentimenti. <br/> 	Foscoliane illusioni estive. <br/> 	Le possibilità non esistono  <br/> 	Venere nata del mare della teoria, dell'astrazione,  <br/> 	dell'ignoranza del tutto.</p>
<p class="indent">
Tiro le reti <br/> 	hai preso qualcosa anche oggi <br/> 	e non hai quanto basta a placare la fame <br/> 	un tormento sopravvivere <br/> 	nella memoria domani, <br/> 	ma non hai neppure il diritto di lamentarti: <br/> 	c'è chi muore accanto a te.</p>
<p class="indent">
Mummia svuotata, giochi di luce alla candela sciolta <br/> 	sensazioni di movimento e apparente immobilità. <br/> 	Ti offro il petto, gli occhi e il naso, <br/> 	generale sparami, prendili e fanne un fantoccio <br/> 	del tuo esercito di vento e acqua <br/> 	che lotti ancora e avverta la terra prima e dopo la battaglia, <br/> 	che abbia nuovi ordini e obbiettivi, <br/> 	limiti.</p>
<h3><a name="9"></a>52 - 26/07/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Viaggio ancora <br/> 	da solo  <br/> 	un mese dopo. <br/> 	Arezzo d'estate <br/> 	nel centro storico di Luglio <br/> 	il giallo delle facciate.</p>
<p class="indent">
Gialle ai lampioni gialli, <br/> 	finzione <br/> 	Passo sbilenco e mani in tasca. <br/> 	Sguardo perduto nell'irrazionale. <br/> 	Finestre aperte, luccicanti di tv <br/> 	discorsi che rimbalzano cadendo dall'alveo familiare.</p>
<p class="indent">
Finestre chiuse, polverose <br/> 	su un'aria ormai vecchia da respirare.</p>
<p class="indent">
Per strada turisti, barboni matti, stranieri, musicisti <br/> 	come me. <br/> 	Cani mi seguono un po' per i vicoli medioevali. <br/> 	Mura massicce conservano squarci di mondo <br/> 	e di tempo di vita.</p>
<p>
Pieve, Piazza Grande, Borg'unto, il Corso Italia.</p>
<p class="indent">
Cammino compiaciuto nel sogno d'estate, <br/> 	scivolo lentamente come la notte dei grilli <br/> 	ad altri pensieri.</p>
<p class="indent">
Al passato recente, che mi ha codificato or ora l'essere <br/> 	in un numero <br/> 	vivere targato. <br/> 	Fredda rugginosa, pratica necessaria, <br/> 	 sintetica etichetta.</p>
<p class="indent">
Al futuro ed a te <br/> 	che domani forse incontrerò per la prima o l'ultima volta <br/> 	appuntamento <br/> 	in cui forse parlerò dell'ennesimo errore. <br/> 	Ricevimi adesso mia regina <br/> 	alla corte delle 11 suonate. <br/> 	Forse non crederai <br/> 	ma ascolterai esperienze ancora vive <br/> 	finché il corpo sarà ancora caldo. </p>
<h3><a name="10"></a>21/07/1997 - 21/07/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">

<h3><a name="11"></a>The al Mattino - 21/07/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Galleggiavo in una brocca <br/> 	di vita bollente <br/> 	e mi bastava. <br/> 	Fluttuavo. <br/> 	L'aroma da me partiva <br/> 	e a me tornava. <br/> 	Arricchito. <br/> 	L'avere tanto sviscerato <br/> 	finalmente sfiorato <br/> 	un compenso di "lunghe fatiche". <br/> 	Già la sera <br/> 	una telefonata si beveva <br/> 	le mie speranze <br/> 	mutilava le mie legittime pretese <br/> 	mortificava i miei sforzi. <br/> 	Crollava il mito del futuro <br/> 	perché il presente  <br/> 	forniva colonne troppo snelle, elastiche <br/> 	per sostenere il passato. <br/> 	In un attimo finiscono mondi <br/> 	ipotetici. <br/> 	Ci si appanna, deprime <br/> 	senza colpevoli <br/> 	solo realtà. <br/> 	La realtà si beveva <br/> 	la vita che mi stavo illudendo. </p>
<h3><a name="12"></a>La fine - 13/07/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ho sognato <br/> 	di essere a Berlino, con i miei, <br/> 	 si va lontano <br/> 	la notte <br/> 	E l'aria era <br/> 	tesa, folle, fragile. <br/> 	Di sera <br/> 	ai piedi di una chiesa <br/> 	là sotto la città <br/> 	brulicante di luci. <br/> 	Un mondo. <br/> 	Nella penombra interna solo qualche sussurro <br/> 	sgranato, monotono. <br/> 	E la fine, che viene, consapevolmente. <br/> 	Lontano, all'orizzonte <br/> 	il bagliore, il tuono <br/> 	immenso di un'esplosione nucleare. <br/> 	Un'altra. <br/> 	Panico e lacrime, la fine. <br/> 	Mio padre guarda e commenta <br/> 	parla di come l'uomo <br/> 	per sua natura <br/> 	inghiotta  <br/> 	se stesso <br/> 	ogni tanto. <br/> 	Ci stendiamo dentro <br/> 	oppressi sul bianco e freddo marmo <br/> 	pavimento di ghiaccio, <br/> 	ad aspettare. <br/> 	Ed ecco grida e follia, la fine. <br/> 	Mi volto in faccia all'inferno <br/> 	fuori la porta. <br/> 	Solo un vento caldo <br/> 	di cenere umana  <br/> 	vi entra. <br/> 	All'esterno arancio, giallo, viola, rosso, troppo <br/> 	Infine silenzio <br/> 	senza grilli. <br/> 	Usciamo in un nuovo mondo. <br/> 	Né luci <br/> 	ma solo lattinea e lontana <br/> 	la luna. <br/> 	Non uomini <br/> 	ma esseri soli, <br/> 	che ripartono dopo la fine.</p>
<h3><a name="13"></a>Sole sul Sale - Puglia - 08/07/1997</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
I) Avvertenze <br/> 	Si viaggia per vari motivi. Vedere un luogo, una città o una cultura. Si viaggia anche per visitare se stessi,  <br/> 	sotto il sole, cocente del sud,  <br/> 	nel luglio caldo e arido, <br/> 	accecante nell'immaginazione e nella realtà.  <br/> 	Sole prosciugatore degli umori illusori e virulenti  <br/> 	dell'inverno intimo e della primavera folle.</p>
<p class="indent">
II) Trun-trun, trun-trun. Trun-trun, trun-trun...trun-trun, trun-trun....acciaio, <br/> 	forgiato nel fuoco rovente  <br/> 	non ti squagli nel torrido pomeriggio <br/> 	di una stazione di provincia. <br/> 	Ti dilati ma non soffochi <br/> 	e per questo ti isoli <br/> 	5° elemento del panorama che arranca <br/> 	e stacca la coscienza <br/> 	per non sopportare oltre <br/> 	il peso <br/> 	di ore immobili <br/> 	ardenti. <br/> 	Paesaggio che diventa dal finestrino di un treno, <br/> 	sempre più alieno, <br/> 	nei tanti anfratti <br/> 	e forme <br/> 	che non percepisco <br/> 	o comunque non ho il tempo di considerare <br/> 	o comunque non conoscerò mai... <br/> 	rotola lento <br/> 	un occhio sul giornale di oggi <br/> 	e una città, un albero, una baracca, un uomo <br/> 	sconosciuti, ignoti mondi <br/> 	se ne vanno da me. <br/> 	Catrame, ribollire, tettoie, binari, folla, sole. <br/> 	Sole <br/> 	cose <br/> 	che  <br/> 	rimangono. <br/> 	Appiccicate alla memoria, pur non essendo mai divenute  <br/> 	protagoniste del viaggio.</p>
