<h2><a name="1"></a>Equivoci orientali - Gennaio 2008</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La luce dell'est  <br/> illumina casa. <br/> Ogni tanto il treno si ferma in luoghi insensati <br/> sfumati dettagli prima scomposti in colori strisciati <br/> tornano elementi, piccoli dimenticati, di un paesaggio <br/> ghiacciato <br/> nel presente. <br/> E appare un sasso, una sigaretta tra i binari. <br/> Tanto nulla per poco vicino esiste. <br/> Nel muoversi del treno il reciproco sguardo si perde. <br/> Ma qualcosa non mi lascia.</p>
<p class="indent">
Ho i tuoi capelli negli occhi e la polvere ghiacciata sopra posata <br/> neve cade da quel cielo impossibile. <br/> Lingua dimenticata sfiorarsi le mani. <br/> Fra noi pesanti cappotti. <br/> Sottotitoli musicali  <br/> per un bellissimo corpo  <br/> oh, come la neve fuori sul davanzale <br/> quale spiegazione è necessaria, <br/> quale testa, pero', mi sono sopra immaginato. <br/> Strana sensazione toccare <br/> esitante con mano tremante <br/> ancora un ricordo una volta <br/> creduto perduto <br/> forse battuto od estinto. <br/> Un altro coraggio <br/> diverso ho trovato  <br/> di piegarmi sul destino <br/> per baciare una bocca <br/> tanto bella.</p>
<ul class="unorderedList">
<li>--------------------</li>
</ul>
<p class="indent">
The east light  <br/> enlighten home. <br/> Sometimes the train stop in senseless places, <br/> faded details before broken into liquid colors <br/> return elements, small and forgotten, of a landscape <br/> frozen <br/> into present. <br/> And a stone appears, a cigarette between rail tracks. <br/> So much nothing for a while exists. <br/> When train moving, mutual look is lost. <br/> But something doesn't leave me.</p>
<p class="indent">
I have your hairs into eyes and frozen dust over them lied <br/> snow falling down from that impossible sky. <br/> Forgotten language is hands's lightly touch. <br/> Between us heavy coats. <br/> Musical subtitles  <br/> for a beautiful body <br/> oh, like snow out on sill <br/> which explanation is needed, <br/> which head, however, i'm imaginating. <br/> Strange sensation is touching <br/> hesitating with shaking hand <br/> another time a memory once  <br/> believed lost <br/> maybe won or extinct. <br/> Another bravery  <br/> different I found <br/> to bend myself on destiny <br/> in order to kiss a so pretty  <br/> mouth.</p>
<h2><a name="2"></a>Libri in Vetrina - Dicembre 2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Libri in vetrina, <br/> un giorno qualcuno entrerà e mi si porterà via. <br/> Ancora sotto alla finestra d'angolo, <br/> Nel giallo del silenzio <br/> Come dieci anni fa ripenso alle cose <br/> che la notte d'agosto suggerisce <br/> sorridendo col dito disteso. <br/> Temevo e sapevo allora di perdere, <br/> e quanto da allora ho perduto. <br/> Quanto già, al termine della mia prima gioventù <br/> avvertivo distante. <br/> Eppure se tutto ciò non avessi perso, cosa avrei trovato. <br/> Pagine perse perché troppe pagine. <br/> Cosi' acquirente scopri il dilemma <br/> se libri ciclici <br/> o vuote superficiali costole <br/> dai titoli infiniti.</p>
<h2><a name="3"></a>Why Not - 03/09/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Cantanti dalle nebbie di autunni anni '70 <br/> spopolano  <br/> la notte terza di fine estate che incombe <br/> come una languida fanciulla al suo pettine. <br/> Cresce in me la voglia di non saper che fare <br/> con le tante tante parole  <br/> faticosamente dette e intensamente credute <br/> che mai cosi' perse  <br/> mi ritrovo a pensare. <br/> Quel vento, mi ricordo bene come forte il suo braccio al collo stringeva, <br/> scirocco e tramontano insieme  <br/> la direzione assoluta pareva indicare. <br/> Da ogni parte arrivai, ma senza essere partito mai. <br/> Dei miei occhi le palpebre si chiusero e m'abbandonai  <br/> ad ammirare lo splendido panorama <br/> di templi e colossi eterni  <br/> torri e alfieri di colui che gioca con se stesso. <br/> Il gioco  <br/> che di partecipanti non bisogna. <br/> E la partita è un lampo insensato <br/> che scocca e brucia due ciechi ubriachi. <br/> Un destino possibile  <br/> palpabile come una tumida zucca  <br/> di marmo appare al tatto <br/> perfetta nell'incastro la forma. <br/> Nulla più è troppo perfetto. <br/> Oggi quel vento ritorna <br/> con identiche parole, <br/> non so se canta la mortalità del vero <br/> o la longevità del falso. <br/> Sono una piccola vela <br/> non più candida <br/> che il vento ascolta. <br/> Perché no, senza capire.</p>
<h2><a name="4"></a>Intercity Notte - 23/08/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
E non so più se per sonno <br/> o se per asfittica mancanza d'aria <br/> cambio ancora stato di coscienza. <br/> Vorrei ricordare <br/> quando già pensai i pensieri stridenti e fischianti  <br/> che si affacciano da fuori al finestrino; <br/> un sogno, un giorno, una vita fa. <br/> E guardano dentro. <br/> Attendo come l'ultimo lampione <br/> perduto nella notte autunnale  <br/> di una zona industriale padana.</p>
<p class="indent">
Crivello e setaccio ricordi recenti e antichi. <br/> Cosa mai importa la loro relativa distanza. <br/> Flash di neon. <br/> Una voce nel corridoio. <br/> L'ennesima stazione <br/> si sofferma incerta oltre il profilo sporco della notte <br/> e negli occhi stanchi  <br/> socchiusi di coloro che mi viaggiano intorno. <br/> Finché il controllore improvviso. Biglietti. <br/> Finché non appare chiaro  <br/> che questa oscurità <br/> non è mancanza di luce ma il presagio <br/> dell'estate morente <br/> che nell'umida terra nasce. <br/> </p>
<h2><a name="5"></a>Contromano nel tempo - 21/07/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Scorre il giorno, poi la sera <br/> e ho perso <br/> la voglia di sorridere <br/> delle mie abituali debolezze. <br/> Forse perché  <br/> la virtù di un'unica parola sussurrata <br/> al mondo per sempre <br/> lontane pose identità fluttuanti <br/> malgrado mille ignorati pensieri <br/> inconsapevolmente contemporanei. <br/> Stessa luna stesse stelle, altro pianeta.</p>
<p class="indent">
Un giorno due vecchi grinzosi <br/> s'incontrarono al bordo del parco <br/> dove accade di arrestarsi <br/> indugiare <br/> incerti indulgere in induzioni <br/> su passi passati. <br/> Intorno gente per caso, solo per caso, <br/> come sempre intorno a loro. <br/> Negli occhi dei vecchi  <br/> rughe non portano più lacrime <br/> commozione e rabbia. <br/> Solo l'emozione di ricordare. <br/> Sono secchi alvei della memoria <br/> pace dei sensi e ribellione della ragione. <br/> Quei vecchi.  <br/> Li odo scontrarsi, <br/> colui che fui e non sarà <br/> o che sarà e non fui. <br/> Li sento dissolversi <br/> come la sera sotto i lampioni.</p>
<h2><a name="6"></a>Pergo - 25/04/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Forse per un cartello con scritto Pergo <br/> forse per problemi di digestione <br/> ho sognato ancora di far l'amore con lei. <br/> E lei passionalmente amava me. <br/> Il vuoto sotto le coperte al mattino <br/> come il climax all'orgasmo <br/> ma molto di più <br/> perché le rondini gridano il suo nome <br/> e in bocca <br/> di quel nome <br/> ho ancora il sapore.</p>
<h2><a name="7"></a>Passignano - 20/04/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Teresa parla ai fiori, <br/> un mattino <br/> tanti anni fa. <br/> Il sole cade obliquo <br/> sulla fronte liscia <br/> di Teresa.</p>
<p class="indent">
Non ci sono spazi vuoti <br/> nella camera di Teresa <br/> affacciata al risveglio <br/> i suoi occhi si specchiano nel lago.</p>
<p class="indent">
La treccia castana di Teresa <br/> la sua bocca e i piedini, <br/> un attimo, <br/> dove ogni cosa è ignara di se.</p>
<p class="indent">
Teresa alla finestra del casale <br/> che si bagna di sole <br/> tanti anni fa <br/> è un ricordo <br/> (Una tortora vola via. Poco lontano) <br/> che non ho mai avuto.</p>
<h2><a name="8"></a>Baricentrale - 10/03/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ricordo guardavo <br/> a questo mare di ulivi che si perde nel mare <br/> asciutto, inquieto <br/> come chi scruta l'imbrunire per credere nell'alba.</p>
<p class="indent">
Da ogni punto che nel mondo ho raggiunto <br/> si è protesa una retta ideale  <br/> rivolta perpendicolarmente altrove, <br/> dove la geometria impossibile  <br/> torna ad intersecare  <br/> i ricordi dell'amore.</p>
<p class="indent">
La pioggia in rivoli <br/> già dal finestrino e <br/> e nei miei pensieri un suono sommesso <br/> di risacca.</p>
<p class="indent">
Nessun luogo  <br/> era lontano <br/> nessun luogo  <br/> era abbastanza vicino. <br/> La mia esistenza <br/> prescindeva dallo spazio <br/> ma nello spazio tracciava le rotte <br/> che ancora adesso <br/> continuo a seguire.</p>
<p class="indent">
Anche tu mi sarai testimone di quanto e dove la cercai.  <br/> In questa città stazione treno vagone sedile <br/> quante volte è passata <br/> vedendomi come adesso vedo lei.</p>
<p class="indent">
Vi sono luoghi <br/> dove gli spiriti si scambiano cortesie <br/> tenereimmmonde.</p>
<p class="indent">
Quell'indifferenza deprecata un tempo <br/> temuta <br/> è apparsa silenziosa a velare l'emozione <br/> come uno sguardo troppo vicino ad una fredda finestra <br/> sotto la quale sono stato  <br/> stavolta senza sperare <br/> né temere.</p>
<h2><a name="9"></a>Kismet (to Feyza) - 24/02/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Sguardo, un raggio tremulo si stacca <br/> dietro spalle voltate il sole tramonta. <br/> Quella neve, e cenere, nello spazio.</p>
<p class="indent">
Nell'attimo in cui la porta <br/> ogni volta <br/> si chiude celando il presente <br/> il passato si manifesta. <br/> Allora palesi paiono le circostanze remote <br/> che mossero da involontari incontri <br/> verso questi strani addii.</p>
<p class="indent">
Doveva cadere anche qui un velo <br/> davanti la mia conoscenza. <br/> </p>
<ul class="unorderedList">
<li>---</li>
</ul>
<p class="indent">
Stare, a tremulous ray detach <br/> behind turned shoulders sun fall down. <br/> That snow, and ash, into space.</p>
<p class="indent">
In the moment which door <br/> every time <br/> is closed hiding present <br/> the past appears. <br/> So evident look the remote circumstances <br/> that came from unwilling encounters <br/> to that strange goodbye.</p>
<p>
Here too a veil must falls <br/> in front of my knowledge.</p>
<h2><a name="10"></a>Le belle donne (to Marteczka) - 14/02/2007</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p>
Le donne attraggono <br/> o ripugnano <br/> le donne che scivolano.</p>
<p class="indent">
Le belle donne attraggono <br/> e ripugnano <br/> quelle che permangono <br/> ombre nei cortili della memoria.</p>
<p class="indent">
Angeli e demoni s'abbracciano <br/> nello sguardo di tali creature.</p>
<p class="indent">
Uomini ciechi vantano di trarle in inganno <br/> ma son le sirene che cantano a loro stesse <br/> l'altrui malizia <br/> e a volte cantano troppo bene.</p>
<p class="indent">
Affacciate su cortili d'ombra <br/> e di tempo <br/> godono lo struggimento di attendere <br/> colui che non le potrà mai avere.</p>
<p class="indent">
E chi le ha, tanto più povero <br/> come della luna <br/> ne possiede meta'.</p>
<p class="indent">
Belle donne, terrore e conforto, <br/> futuro e passato, presente giammai...</p>
<p class="indent">
La più bella di tutte <br/> ha un amore paziente, <br/> tornerà a sussurrare ancora <br/> nenie di culla.</p>
<p>
La più bella di tutte  <br/> stupenda, attende anche me.</p>
<ul class="unorderedList">
<li>--</li>
</ul>
<p>
Women are charming <br/> or are nauseating <br/> Women that drifting.</p>
<p class="indent">
Beautiful Women are charming <br/> and are nauseating, <br/> these that remain <br/> shadows into memory's courtyards.</p>
<p>
Angels and daemons hug <br/> into the look of these creatures.</p>
<p class="indent">
Some stupid man are boasting that they can lead on them <br/> but sirens sing to themselves <br/> other people deceit <br/> and sometimes they sing too well.</p>
<p class="indent">
While they overlook into shadow and time courtyards  <br/> they enjoy the distressing of waiting <br/> who never may have them. <br/> </p>
<p class="indent">
And who has them, poor him, <br/> as the moon <br/> he has only the half part.</p>
<p class="indent">
Beautiful women, terror and comfort, <br/> future and past, never present...</p>
<p class="indent">
The best one of them <br/> has a patient love, <br/> she will return to whisper <br/> craddles lullaby.</p>
<p>
The best one of them, <br/> wonderful, is waiting me too.</p>
<h2><a name="11"></a>Non si fanno prigionieri - 04/12/2006</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Spazio nuovo, aria aperta <br/> liberi occhi su libere mani. <br/> Mai più prigionieri, mai più farò. <br/> Né fate né gnomi, villani o re <br/> nessuno di loro risparmierò. <br/> Cadranno tali a gocce appese alla foglia <br/> senza conoscere rancore, <br/> dimentichi di me come di una nuvola notturna. <br/> Non ambiranno a cadere ma cadranno <br/> nel soffio di zefiro o col volo d'un'ape. <br/> E la mia ombra dall'alto nella fronda <br/> già immagino nascosta <br/> tra le pieghe della notte, <br/> nei silenzi di vite dove <br/> per nessuno sarà necessario. <br/> Se amo, d'amore <br/> mai più farò prigionieri.</p>
<h2><a name="12"></a>L'indole del Gatto (to Emmy) - 11/11/2006</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Si abbandonano alle volte ciecamente <br/> tra le braccia <br/> e si rivoltano in altri momenti <br/> per graffiarti <br/> al solo tuo appressarti. <br/> Cosa pensano, cosa vedono non ti è dato sapere <br/> ma pare sentano ciò che non esiste. <br/> E' cosi' facile pestare una delle loro code <br/> o non trovarne più nessuna <br/> nella mutevolezza della loro apparenza, <br/> mentre è celata la loro vera natura che, <br/> s'intende, <br/> più vera non potrebbe essere, <br/> qualunque essa sia.</p>
<p class="indent">
Ciò da cui provieni e verso dove proseguirai <br/> nei loro occhi soltanto <br/> è semplice leggere, <br/> occhi che ti guardano mentre non vedi.</p>
<p class="indent">
Camminano su nuvole che loro stesse generano sognando. <br/> Solide nuvole.</p>
<p class="indent">
Nel passo silenzioso e nel modo di voltarsi conosco la cura nell'invadere <br/> e la curiosità di scoprire tutto quel che non è evidente, <br/> quasi fosse cosa ormai troppo nota <br/> la propria foggia, da dovervi tornare.</p>
<p class="indent">
Riconoscenza e legame non sono concetti che subiscono <br/> ma che forgiano ogni volta che la loro mente li concepisce <br/> capaci cosi' di rinnovarli in eterno e subitaneamente rinnegarli.</p>
<p class="indent">
E quando giocano lo fanno per disperazione <br/> la paura di aver dimenticato.</p>
<h2><a name="13"></a>Montecchio - Aprile 2006</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Il cuore gonfio <br/> il cuore grande <br/> come il cielo riflesso <br/> negli occhi dei cani. <br/> Mi imbratto di un semplice sorriso <br/> tolgo la giacca e seguo qualcosa, <br/> sale lungo fili d'erba <br/> che del verde fanno innocenza.</p>
<p class="indent">
Mi sovrasta un castello, in questo spazio leggero. <br/> Un istante azzurro una volta mio <br/> è mio perdutamente. <br/> Dal silenzio della solitudine cado <br/> nell'euforia di quel battito lieve <br/> che si apre al cuore quando il polso s'acquieta. <br/> Un reciproco ascolto, dove nessuno mi sente.</p>
<p class="indent">
Ora tutto s'invola col vento d'aprile. <br/> Tanti pensieri cari, come aquiloni. <br/> Ci sarà un mondo, oltre la strada <br/> al di la della curva che nega ogni certezza <br/> m'è facile indovinare questa mia dolce campagna <br/> continuare.</p>
