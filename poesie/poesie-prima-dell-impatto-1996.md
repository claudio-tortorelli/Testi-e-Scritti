<h2 style="margin-top:40px;">Introduzione</h2>
<p class="indent">L'immagine di una roccia che si avvicina inesorabilmente alla Terra, portando una distruzione totale, fa sorgere un processo di riflessione. La fine che diventa reale, attraverso una cosa reale come un asteroide. La fine che diventa inevitabile, definita e sicura. Di fronte alla fine sorgono alcune domande, si arriva a conclusioni non considerate prima, ma soprattutto scompaiono le piccolezze umane, le verità imposte, per far posto a nuove verità dettate dalla coscienza, dalla paura, dal sentimento o da come lo si vuol chiamare. Chi c'è dietro la bomba che sta per cadere, se c'è qualcuno? Questa roccia, questa ipotetica fine, non è che l'emblema della debolezza dell'uomo, che non può opporsi alla morte, ma che, come diceva Pascal, è cosciente di ciò.</p>

<h2><a name="1"></a>La scienza - 14/11/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ecco l'onda, il segnale <br/> che torna costante <br/> sul monitor verde-piatto <br/> lascia l'occhio stanco, <br/> non solo. <br/> Non più <br/> equivochi sul futuro <br/> certo. <br/> Già la mia mente <br/> che ho coltivato <br/> è pronta a rendere questo corpo affittato. <br/> Sarò pronto allo sfratto <br/> penso al domani-nulla <br/> riso e pianto <br/> si seccano sul volto teso, <br/> in una maschera. <br/> Quante persone <br/> quanti attimi <br/> quanti eventi <br/> sprecati già una volta <br/> si perderanno domani, <br/> in un rosso bagliore <br/> che solo io abbraccerò <br/> in piedi. <br/> Le epoche <br/> gli uomini, i sentimenti, gli amori <br/> si sono aggrovigliati <br/> per finire così? <br/> A metà...? <br/> Non so ancora <br/> quando ebbero inizio, <br/> e devo già conoscerne la fine. <br/> La scienza... <br/> quale fortuna è più grande <br/> di quella di poter <br/> saper disporre di un <br/> tempo finito? <br/> Guarda là fuori, <br/> dalla finestra <br/> un mondo <br/> finalmente sensato <br/> davanti alla fine... <br/> sono io che non capisco.</p>
<h2><a name="2"></a>La giovinezza - 20/11/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Passa, con andatura impegnata <br/> nella ricerca  <br/> di un'imitazione perfetta. <br/> Un ragazzo guarda <br/> davanti a se il vento <br/> scuote i suoi capelli. <br/> A cosa pensa? <br/> Perché non correre fuori e dirgli: <br/> "Non ti preoccupare più <br/> tu non lasci la tua giovinezza, <br/> non la perderai <br/> nel vivere", <br/> per poi non leggergli negli occhi <br/> una fiamma che <br/> giustamente si spegne, <br/> che non potrà lasciare fumo <br/> né cenere. <br/> Si vive per imparare a morire <br/> io <br/> ancora devo imparare a vivere.</p>
<h2><a name="3"></a>La musica - 24/11/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Corde di chitarra <br/> son le tue ciglia <br/> sospese e tirate <br/> su un occhio che guarda <br/> fisso a qualcosa di non presente, <br/> vibrano al vento, <br/> forse sentono, forse sanno. <br/> Quelle mani <br/> hanno sfiorato rumori <br/> quella voce <br/> ha plasmato la creta <br/> che il cuore trasuda. <br/> E quanti sono riusciti a sentirti, a viverti. <br/> Non sei vissuto solo <br/> e sei vissuto in 100 vite. <br/> Ma ora dimmi <br/> dove andrai a vivere <br/> in quale mondo <br/> universo <br/> civiltà tornerai, <br/> tra quanto tempo <br/> in quale anima <br/> sarò, dovrò <br/> aspettarti ancora.</p>
<h2><a name="4"></a>La vecchiaia - 27/11/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Vecchio tu appari <br/> e forse lo sei. <br/> Quando trascini i tuoi legni <br/> giù dal letto, <br/> quando il cuore dimentica <br/> ignora un nuovo mattino, <br/> quando preghi <br/> in silenzio <br/> il Dio della morte-lampo. <br/> Ma quando invece <br/> questo non accade, <br/> quando il vento <br/> ti apre le finestre <br/> su una nuova primavera, <br/> sei vecchio anche allora? <br/> Cosa hai imparato? <br/> Entrambi adesso <br/> vediamo questo <br/> presente grigio, <br/> ma tu hai visto anche <br/> quello che io non vedrò. <br/> Dio della vita-lampo, <br/> sono stato vecchio <br/> nel cercarti, <br/> adesso mi sentirei giovane; <br/> di fronte alla morte  <br/> si capisce l'età.</p>
<h2><a name="5"></a>L'amore incompreso - 01/12/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Era di là <br/> dormiva ancora <br/> in un'altra stanza. <br/> Si conosce <br/> al primo sguardo <br/> una persona, una donna, un'anima, <br/> più alta, <br/> come invece ci si accorge <br/> di non saper nulla <br/> dell'amore che non sia miraggio mentale, <br/> di chi si tenta di abbracciare <br/> come nuvola di vapore. <br/> Domenica mattina <br/> per la mia vita, <br/> mentre il sole delle 9 <br/> dopo giorni di autunno <br/> entra dal ricamo delle tende, <br/> illumina <br/> finalmente <br/> leggo anche <br/> sul cassetto del comò <br/> bagnato di luce <br/> lo spegnersi, il riaccendersi <br/> il passare delle nuvole. <br/> Da dove è partito quel raggio? <br/> Allora non capivo, <br/> sentivo un pianoforte <br/> ma lo sentivo suonare lontano. <br/> Oggi che sto per partire <br/> mi accorgo di cosa ho già lasciato <br/> altrove, <br/> per voler rimanere, <br/> in questa strana malinconia sorridente.</p>
<h2><a name="6"></a>L'odio - 06/12/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Non ha un volto <br/> verso cui concentrare il pugno <br/> solo un ghigno <br/> imperdonabile. <br/> Rivive ancora <br/> dietro il sipario mentale <br/> la tragicomica <br/> un'intollerabile presenza <br/> che non c'è, <br/> ed eccitarsi di nuovo <br/> alla soddisfazione <br/> del nostro, personale, finale. <br/> Perdonare... <br/> Troppe volte <br/> ho perdonato me stesso <br/> prima di perdonare gli altri, <br/> odiato gli altri, <br/> prima di imparare a odiare me stesso.</p>
<h2><a name="7"></a>Lo spirito assoluto - 09/12/96</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ma le ginestre <br/> Ma le ginestre <br/> dove sono <br/> cosa bevono <br/> di inverno <br/> d'estate <br/> nel campo? <br/> nel sole? <br/> Se non esisti, ho paura. <br/> Se ci sei, ad aspettarmi, ne ho di più. <br/> Rivedo giorni, sprecati nel bello <br/> quando c'era tanto brutto da apprezzare! <br/> Risento vibrazioni di gioia, <br/> dopo aver costretto ondate di dolore, in una lacrima. <br/> E in mezzo a questo <br/> c'eri Tu? <br/> Qualcuno muore nel pronunciarti. <br/> Qualcuno brucia nel negarti. <br/> Ed io ho sorriso colpevole ad entrambi. <br/> Tra poco sarà buio? <br/> Se lo sarà non me ne accorgerò, <br/> e questo è il conforto della scienza. <br/> Tra poco sarà luce? <br/> Se lo sarà, sarai Tu l'ingiusto <br/> che ammette alla luce chi <br/> per guidare gli uomini, <br/> ha acceso candele. <br/> Lascio la vita.</p>
<h2><a name="8"></a>Boom Boom - 31/03/97</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Due volte lo schianto <br/> mentre entra nell'atmosfera <br/> lacera la mia mente <br/> la mia forza si disperde <br/> sono un uomo <br/> sotto un cielo rosso <br/> il sangue vola via <br/> questione di attimi infiniti <br/> se il tempo è veramente uno scrigno <br/> inviolabile. <br/> Caldo, vibra la vista al lampo viola.</p>
