<h2><a name="1"></a>02/09/98 - 02/09/98</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Quello che ero <br/> io odierò <br/> perché si odia <br/> quello che non si comprende. <br/> E i miei sogni puerili <br/> i patetici Amori da solleone <br/> in cui il cuore stride <br/> al calpestar del pensiero <br/> sono stati intimamente ripudiati. <br/> Troppa la delusione <br/> troppo il rimpianto <br/> troppo tempo <br/> in pochi giorni. <br/> Io non riesco più ad estasiarmi <br/> in riflessioni a spirale. <br/> Quello che mi riempiva il cielo, <br/> un sentimento o una stagione, <br/> una fantasia <br/> ora <br/> non basta più. <br/> Accolgo giorni e ore e minuti <br/> aridamente <br/> assiduo ormai alla povera disciplina <br/> dell'uomo.</p>
<h2><a name="2"></a>15/10/98 - 15/10/98</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ad Ottobre <br/> tira il vento <br/> che non porta parole <br/> né campane. <br/> Aria silenziosa <br/> come una foglia che si stacca, che tocca terra. <br/> Ottobre è un sogno <br/> di un sonno profondo <br/> da cui non ti svegli. <br/> E' un ragno <br/> che geometricamente ti avvolge. <br/> Un treno che parte <br/> che non si raggiunge <br/> ti porta a vivere altrove. <br/> Anche se non vuoi, ormai ci sei. <br/> La vita cambia colore ad Ottobre <br/> e anche l'albero sotto il sole, <br/> ricco di luce, <br/> si secca in una notte  <br/> d'Ottobre. <br/> Non bastano parole a chi cercava riparo <br/> da un altro inverno <br/> nei lunghi mesi di gelo <br/> cosa li scalderà? <br/> Ad Ottobre c'è un mondo che muore <br/> ogni anno <br/> senza farsene accorgere. <br/> Ad Ottobre non tornano rondini ai tetti <br/> e la fede è una gomena sciolta <br/> dopo la tempesta. <br/> In Ottobre nel cielo corrono veli <br/> o sono forse sugli occhi <br/> quelle lacrime, di un incredulo dolore.</p>
<h2><a name="3"></a>30/10/98 - 30/10/98</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Una volta <br/> ho visto una nuvola <br/> ed un uccello fuggirle <br/> impazzito <br/> poi rincorrerla. <br/> Una volta ho ammirato un tramonto <br/> e i colori della campagna <br/> nell'albeggiare della luna, incenerirsi. <br/> Ho visto e udito, <br/> tutto quello che ho provato <br/> è stato <br/> una volta <br/> una volta sola. <br/> Dimmi, amica, se tu hai adesso un senso cosmico <br/> una percezione universale <br/> che fugga ed eviti lo scippo interminabile <br/> del tempo e della vita, sensazioni perdute, <br/> che consideri questo Ottobre <br/> la fine e l'inizio <br/> di un sistema illimitato e ricorsivo. <br/> E noi variabili <br/> ripetibili <br/> sostituibili.</p>
<h2><a name="4"></a>31/10/98 - 31/10/98</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Fumo, tanto e denso <br/> nuvola nella nebbia, <br/> così ricordo la partenza. <br/> Poi il blu, sempre più scuro <br/> nero <br/> e spazio dinamico intorno alla staticità, <br/> insondabile. <br/> Nei cieli siderali ho viaggiato <br/> imparando la violenza del nulla invisibile e l'ombra. <br/> Imparando ad invecchiare, alla velocità della luce. <br/> Intorno a quanti corpi celesti ho orbitato <br/> non posso ricordare, <br/> non si contempla l'infinito. <br/> Attorno a piccoli pianeti <br/> dalla grande forza attrattiva, <br/> agli immensi giganti gassosi, <br/> giocando alle volte con i loro satelliti, <br/> sono diventato una meteora nel Caos <br/> di un anello.</p>
<p class="indent">
Di tanto in tanto una cometa. <br/> Poi da ognuno, <br/> grazie alla fisica inesorabile del tempo, <br/> sono sfuggito</p>
<p>
... ho viaggiato ...</p>
<p class="indent">
ed ognuno è adesso lontano. <br/> Alle volte andando incontro al nulla, <br/> mi imbattevo in esseri senza luce <br/> ma molto più grandi, profondi, <br/> sembravano inesistenti <br/> ma sono loro a cambiare l'universo. <br/> Solo un po' di luminosità <br/> gli ho potuto regalare. <br/> E loro in cambio mi hanno mostrato <br/> come non esista il vuoto e la distanza.</p>
<p class="indent">
E' stata questa la mia vita. <br/> E' stato solo questo giorno. <br/> Un giorno di Ottobre. <br/> Uno qualsiasi.</p>
<h2><a name="5"></a>03/11/98 - 03/11/98</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Come la luce <br/> scende a Novembre <br/> cullando i bambini <br/> sull'altalena, <br/> nel parco giallo e rosso <br/> su sfondo grigio <br/> ritorno col pensiero di ieri <br/> che oggi ricordo. <br/> E' una lettera fra le mani <br/> il lasciapassare <br/> che aspettavo da tempo. <br/> Sono su un treno <br/> destinazione ignota. <br/> Colgo distratto <br/> immagini in movimento, <br/> funghi e castagne <br/> nel sottobosco mi attende <br/> l'umido e il violoncello <br/> che il cuore accompagna <br/> a solitarie <br/> autunnali peregrinazioni. <br/> La Lettera taglia <br/> un attimo solo <br/> la nebbia pensierosa <br/> d'autunno.</p>
<h2><a name="6"></a>Clessidra - 28/11/98</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La sabbia è molta <br/> e dolce è il tormento <br/> di qualche granello che si perde <br/> che lesto scorre <br/> in un oscuro buco nero. <br/> Non si vede nulla <br/> di lassù. <br/> Si dice sia nostro padre. <br/> Si ipotizza che è nostro figlio. <br/> Intanto altri granelli, lungo la schiena di Crono. <br/> Finché una piccola fossa <br/> si forma al centro dei nostri pensieri <br/> e come un gorgo, o mitico Maelstrom <br/> si fende e si espande inghiottendo, <br/> allargandosi sempre più, nella sabbia. <br/> Ci accorgiamo allora di quanti pugni <br/> di polvere preziosa possediamo, <br/> ed è sempre troppo poca, <br/> sempre meno e il vortice ingoia <br/> anche adesso, ti vuole. <br/> Giù sempre più vicini al buco, <br/> nell'affanno della nostra leggerezza <br/> ormai incommensurabile. <br/> E tanto più i granelli cadono <br/> tanto più impiegano a cadere <br/> ma sono pochi, gli ultimi. <br/> E quando infine <br/> pensavamo che l'ultimo sostegno <br/> cedesse <br/> scivolando giù insieme a noi, <br/> tutto si è fermato. <br/> Buio e silenzio. <br/> Dove siamo? Dove eravamo? <br/> Qualcuno ha ribaltato la clessidra. <br/> Non esiste posizione, <br/> orientamento, <br/> sequenzialità, <br/> disordine, <br/> cronologia. <br/> Per noi il tempo è ricominciato.</p>
<h2><a name="7"></a>Amore perduto - 12/12/98</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Non ti scorderò <br/> perché sei nato e morto un giorno di maggio <br/> la sera tornando a casa <br/> si avverte l'infinito. <br/> Di cosa è fatto il destino <br/> che mi avvolge <br/> che mi lega al percorso stretto <br/> tra infinite modalità di accesso <br/> non trovo che un pomeriggio di umidità d'anima. <br/> In questo pomeriggio <br/> scopro che forse in un punto dello spazio <br/> esiste un amore <br/> che forse incontrerò <br/> di nuovo. <br/> Ella mi aspetta, sul dolce viso, l'alone della serenità <br/> l'azzurro le copre come un velo i capelli <br/> sciolti <br/> pendenti ai seni <br/> solamente accennati <br/> nello sguardo, la vera femminilità. <br/> D'un tratto cammina e ascolta e dice <br/> e il tempo non la costringe <br/> talvolta anzi si ferma e mi sorride <br/> e colgo un attimo <br/> di felicità, <br/> la fresca primavera dell'anima, <br/> sboccia, <br/> non gli importa se tutto questo <br/> è folle illusione o pallida profezia <br/> crede e spera. <br/> Ma il tempo che lei non avverte <br/> mi separa dal suo tatto e la luce si spegne ancora. <br/> Cala la notte e solo l'amore perduto <br/> mi accompagna nel sogno.</p>
<h2><a name="8"></a>All'ombra - 12/01/99</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
D'un raggio di luce <br/> nella penombra tagliente <br/> come una lama <br/> trapassa il mio cuore <br/> che non sente e non muore, <br/> ho il ricordo, <br/> di quello che non è mai stato <br/> l'emozione. <br/> E di questo e di tutte le disgrazie umane <br/> penso alle volte <br/> e non trovo che la momentanea soluzione di circostanza <br/> che abbraccia la terra ma non il cielo <br/> vola nel cielo ma non tocca mai terra. <br/> Se questo, Signore, fa parte di me, <br/> distruggilo <br/> e crealo nuovo dal fuoco. <br/> Come io ho fatto <br/> Pazzo <br/> accostando nuovamente le persiane <br/> alla luce mattutina.</p>
