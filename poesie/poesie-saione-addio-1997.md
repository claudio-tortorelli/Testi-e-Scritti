<h2><a name="1"></a>Sconfitta - 07/05/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Brucia <br/> nei polpacci e negli occhi <br/> arde <br/> tra denti stretti <br/> morde <br/> il respiro <br/> chiuse <br/> le mani. <br/> (un calcio nell'aria) <br/> Scema <br/> nelle spalle, giù per il collo <br/> crolla <br/> basso sui piedi, lo sguardo <br/> che si apre <br/> come le mani <br/> sporche di fango <br/> le guance <br/> ancora contratte. <br/> Tra i capelli <br/> scompigliati <br/> acqua e sale. <br/> A casa.</p>
<h2><a name="2"></a>03/05/1997 - 03/05/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Il Marciapiede è sempre più stretto <br/> e le macchine passano <br/> fredde <br/> anche in questo pomeriggio rosa-ambra <br/> tendente al malinconico, <br/> come sto muro <br/> come il pisciacane giallo <br/> sotto il semaforo <br/> come il lampione <br/> sopra la cabina, <br/> ....o le mattonelle saltate, <br/> chi le avrà fissate? <br/> Anche la malinconia <br/> ha una casa in città, <br/> e paga l'affitto <br/> per dimorare nel cuore, <br/> la tassa per trasferirsi <br/> dalla materia all'astratto della mente. <br/> Rimani pure e continuiamo la passeggiata. <br/> Al Duomo vecchio o S.Francesco <br/> sotto i portici o per Borg'Unto <br/> aleggio <br/> fantasma pomeridiano <br/> che vive <br/> di rimbalzi di sensazione <br/> nutrendosi di odori e profumi <br/> di donne sconosciute. <br/> Occhi e bocca formano un cerchio <br/> quando il naso respira malinconia. <br/> Poi il sipario si chiude <br/> mentre la trama <br/> evapora. <br/> E da solo scendo le scale. <br/> Metto un piede in una pozza <br/> tra un po' asciutta: <br/> l'acqua terrosa torna ad adeguarsi <br/> all'immagine di un vecchio amico.</p>
<h2><a name="3"></a>Arezzo, la Pieve - 27/04/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Sul tetto, <br/> basso quadrato, <br/> spunta una breve sciabola <br/> di ferro rugginoso <br/> domina incontrastata <br/> la sua nobile solitudine. <br/> Quanti astri sopra la testa <br/> sono ruotati di giorno e di notte <br/> d'estate e inverno <br/> sempre ignorata. <br/> Due monti, quattro valli, una città <br/> di là si vedono mutare pigmento <br/> pulsare di vita e di morte. <br/> Vedere i tetti dove vivono gli uccelli, <br/> essere più lontani da terra <br/> più vicini ai cieli azzurri di Aprile, <br/> a quelli rossi di Luglio <br/> a quelli argento-notte di Ottobre. <br/> Da secoli. <br/> Poco più giù  <br/> pietre serene <br/> allegre al sole <br/> malinconiche alla pioggia <br/> scivolano <br/> sempre più in basso, <br/> arrotolandosi, aggrappandosi a colonne sempre diverse, <br/> tutte uguali nel giallo dei lampioni. <br/> Appoggio l'orecchio al portone <br/> e ascolto <br/> un'eco sempre più lontana, <br/> la mano corre lungo le crepe delle pareti, <br/> zuppe di vernice, storia e piscio. <br/> Un cane abbassa la zampa e svolta l'angolo...</p>
<h2><a name="4"></a>Letto - 26/04/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Mattina <br/> luce gialla rosa dalle tende <br/> filtra il cielo sereno <br/> ancora non caldo. <br/> Caldo è invece il mio letto, <br/> rampa di lancio <br/> allestita ogni sera. <br/> La sera, quando il corpo <br/> cede controvoglia ad una necessità <br/> apparentemente inutile. <br/> Ma sono inutili tutti i graffi, <br/> le smorfie, le carezze <br/> che ruotano la notte <br/> sopra il tempio di Morfeo? <br/> A cosa serve morire e rinascere <br/> altrove  <br/> e morire ancora <br/> se non a placare la sete dell'ordine del disordine <br/> dell'umana onnipotenza e del divino sentimento? <br/> L'inganno del sogno <br/> mi salva dal nero dei miei pensieri <br/> dalla coscienza notturna <br/> già immersa nella luce amica e tonica <br/> della mattina.</p>
<h2><a name="5"></a>Tupakamaru - 18/04/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Sulle dita di una mano aperta <br/> saltano in 15, <br/> evitando con cura <br/> un palmo arido o umido, sporco, <br/> una linea della vita troppo breve, <br/> una fortuna consumata. <br/> Saltano sulle dita <br/> per sfuggire al palmo <br/> freddo e appiccicoso, <br/> dove la morte li attende lenti, <br/> dove a morire si va in tanti. <br/> Ma si muore anche cadendo <br/> dal palmo, tra le dita, <br/> che ti vogliono afferrare.</p>
<h2><a name="6"></a>Salto - 13/04/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ti lascio. <br/> La mano <br/> colpita ricade all'indietro, <br/> la bocca <br/> si chiude: è finito l'affanno. <br/> Si spegne.... <br/> oro placcato sui monti <br/> dove sei nata <br/> muore il giorno <br/> ti accompagna lontano.</p>
<h2><a name="7"></a>Tulipani - 06/04/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Dieci giorni  <br/> per sentire nell'aria l'olezzo di vita <br/> che sboccia <br/> dieci volte al mattino, <br/> se ne chiudono 11 alla sera <br/> di tramonti. <br/> Cielo d'Aprile non mi lasciare <br/> sotto un velo opaco <br/> fine ricamo <br/> e pesante coltrone <br/> di tutti i miei momenti buttati <br/> al tuo vento <br/> troppo troppo veloce <br/> per inseguirlo. <br/> Mi sta scippando la giovinezza <br/> le unghie sono ormai consumate <br/> e perdo <br/> ovunque <br/> un po' di me, <br/> e perdo. <br/> Sono un perdente <br/> che costruisce la sua sconfitta <br/> con coscienza, <br/> per soffrire e rendersi bene conto <br/> del proprio misero stato. <br/> O sono un qualunque <br/> computer da riformattare: <br/> virus d'adolescenza. <br/> Grazie del pensiero comunque <br/> voi tutti che avete popolato il mio cuore. <br/> Ma affondo <br/> e perdo una partita che non ho voluto <br/> vincere. <br/> Resto a guardare lo specchio  <br/> in cui si è incisa <br/> impressa la mia anima, <br/> malinconicamente tormentata <br/> e aspetto <br/> i primi papaveri. <br/> Dei vostri colori tocco <br/> un'ultima volta <br/> l'essenza della mia infanzia.</p>
<h2><a name="8"></a>Garbasso - 30/03/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Su quel letto chi avrà respirato prima? <br/> Di quei respiri è unto il soffitto <br/> bianco <br/> che fissi oltre. <br/> Lamenti <br/> che infrangono il proprio pudore, <br/> incosciente dolore. <br/> Il materasso sibila un duetto <br/> con l'acqua dell'ossigeno <br/> che ribolle <br/> e la flebo scende  <br/> piano  <br/> in un braccio morto e floscio, <br/> o in un piede. <br/> Passa la barella <br/> cigolante <br/> dell'ultimo viaggio <br/> urla <br/> al letto accanto <br/> la pena <br/> e le porte e i muri,  <br/> è caduto l'intonaco, <br/> l'illusione di bellezza <br/> di vita, di splendore, <br/> rimane il Garbasso <br/> ultima stanza  <br/> dove osservare un'anima cara crepare, <br/> un lampione rugginoso <br/> nel freddo della notte alberata <br/> si spegne. <br/> Ma è la notte di Pasqua <br/> vigilia del sole, del cielo limpido <br/> di Marzo, <br/> dell'odore di pollo <br/> che per le strade sconfigge i gas di scarico, <br/> delle benedizioni dalla Tv, <br/> della messa solenne, <br/> dei bambini che si agitano. <br/> La Pasqua.</p>
<h2><a name="9"></a>GRAFFI - 09/03/1997 - 30/03/1997</h2>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<h3><a name="10"></a>Prima</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
E' questo l'attimo! <br/> 	In cui tutto può ancora essere contenuto <br/> 	perché è vago <br/> 	il futuro. <br/> 	La vigilia è un magico forziere <br/> 	per la fantasia non ancora mortificata <br/> 	e mi tuffo, mentre tutto s'allontana <br/> 	in viaggi lunghissimi. <br/> 	E così ogni attimo <br/> 	sarà un viaggio <br/> 	mentre il Viaggio diventerà saturo <br/> 	di memoria. <br/> 	Quante ore avrò da vivere <br/> 	non importa <br/> 	se saprò dividere il tempo infinito <br/> 	in tanti piccoli viaggi <br/> 	non partirò mai.</p>
<h3><a name="11"></a>Salisburgo</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
La prima sera <br/> 	è sempre l'ultima del sogno <br/> 	spremuto e succhiato <br/> 	permane un sapore <br/> 	quando il succo è ingoiato <br/> 	e affascina nel profondo <br/> 	la realtà dell'aroma <br/> 	che lascia l'alone <br/> 	del caldo ricordo <br/> 	sulla fredda memoria <br/> 	domani. <br/> 	La notte sottobraccio sul ponte <br/> 	sul fiume della speranza <br/> 	troppo basso per arrischiarsi al tuffo <br/> 	ma così forte, finché la neve si scioglie. <br/> 	Domani sarò ancora giovane. <br/> 	E le risa e l'avventura <br/> 	demoliscono l'ombra di casa mia <br/> 	lontana, <br/> 	mentre le note qui sono vicine nello spazio <br/> 	che ancora si sentono, <br/> 	dietro i pesanti mattoni <br/> 	che intrappolano gioia e abbandono.</p>
<h3><a name="12"></a>Campagna praghese</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Onde e altre onde <br/> 	da lontano l'eco <br/> 	delle Alpi non si sente <br/> 	nella bonaccia. <br/> 	Il fosso rasato <br/> 	appare e scompare <br/> 	come le rade misere case <br/> 	dietro l'alone del mio respiro <br/> 	intermittente sul vetro. <br/> 	Lontano mi sento un punto. <br/> 	Campi e ciuffi di boschi <br/> 	filtrano l'aria di est <br/> 	mentre un bambino chissà <br/> 	chi è, cosa sarà <br/> 	torna a casa <br/> 	quale casa <br/> 	un'altra casa <br/> 	un po' smurata. <br/> 	E i dettagli vicini <br/> 	sfumano all'occhio <br/> 	mentre scivolano dalla cresta del colle <br/> 	al passato recente, a quello remoto, al nulla. <br/> 	E i dettagli lontani <br/> 	invece lo stesso <br/> 	appiccicano la mente a collose riflessioni <br/> 	perse. <br/> 	E nel mezzo c'è solo  <br/> 	la campagna di Praga <br/> 	l'ondulata, placida, attesa.</p>
<h3><a name="13"></a>Motel</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Neon freddo e umido <br/> 	nella notte ti immagino <br/> 	acceso anche quando non ci sono <br/> 	non ti penso, a illuminare <br/> 	invitare qualcuno nelle due stanze <br/> 	di legno e di vetro <br/> 	sorde al grido della freschezza <br/> 	che passa continua in strada <br/> 	e passive <br/> 	e vecchie <br/> 	alla vita delle persone <br/> 	quante e strane persone <br/> 	che si affacciano diverse nell'occhio <br/> 	uguali nell'ombra <br/> 	che attacchi alla parete. <br/> 	Fuggono via.</p>
<h3><a name="14"></a>Praga la notte</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Praga la notte <br/> 	non la conosco <br/> 	se non nel volto duro <br/> 	che mostra ai turisti distratti <br/> 	illusi di leggere un suo rilassato sorriso <br/> 	sotto i lampioni. <br/> 	Passaggio sul ponte <br/> 	lento e sospeso sul sogno <br/> 	che scorre <br/> 	in un luogo diverso. <br/> 	Le case, le statue, i bastioni <br/> 	me stesso  <br/> 	giù sul fiume <br/> 	ondeggia confusamente <br/> 	al passare della chiatta. <br/> 	Giovani ubriachi di che, <br/> 	vanno cantando e ridendo <br/> 	di una fumosa incoscienza <br/> 	innocenza. <br/> 	Un tram <br/> 	scende un vecchio <br/> 	chiude le porte e se ne va, <br/> 	quando un violinista assonnato <br/> 	conta gli spiccioli nel cappello. <br/> 	Un ragazzo  <br/> 	di spalle dipinge <br/> 	un'altra nottata di nero. <br/> 	Dove la trovo <br/> 	ne accusa l'anima. <br/> 	E' una città diversa <br/> 	una vita diversa <br/> 	così lontana così vicina <br/> 	di notte.</p>
<h3><a name="15"></a>Skoda</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Di ogni chiazza <br/> 	leggi un evento <br/> 	colorato ricordo <br/> 	sulla carrozzeria <br/> 	vecchia, superata, resistente <br/> 	automobile <br/> 	arlecchini daltonici <br/> 	fumano e tossiscono <br/> 	come i padroni dagli occhi asciutti <br/> 	come il tempo <br/> 	cielo passato da lontano nuvola <br/> 	e va avanti <br/> 	con poca benzina.</p>
<h3><a name="16"></a>Ritorno</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Quanto si perde <br/> 	nell'intestino del tempo <br/> 	di un lampo di vita <br/> 	che ti acceca <br/> 	da non vedere più <br/> 	quegli splendidi particolari vissuti <br/> 	quei nomi <br/> 	quei luoghi <br/> 	che sono stati presenti con te <br/> 	sull'atto X della commedia. <br/> 	Ritorno nello stanco viaggio di un pullman <br/> 	nella malinconica notte <br/> 	che è inevitabilmente l'ultima  <br/> 	di queste <br/> 	chiaramente solo un ricordo. <br/> 	Ritorno con le tasche piene di frammenti <br/> 	di un'esperienza strana <br/> 	claustrofobicamente unica; <br/> 	e le tasche bucate seminano e perdono. <br/> 	Ritorno con speranze in cui si credeva <br/> 	per noia dimenticate <br/> 	stanco. <br/> 	Ritorno e la droga circola <br/> 	ad ampie boccate <br/> 	il neon verde, rossi e bianchi fanali <br/> 	colori della penombra, <br/> 	del dormiveglia della coscienza <br/> 	strozzata, <br/> 	cianotica guarda  <br/> 	sempre più spenta <br/> 	volti luoghi sentimenti, <br/> 	lontani <br/> 	perdersi <br/> 	nel reticolo spazio-tempo. <br/> 	Tela del ragno Normalità.</p>
