<h3><a name="2"></a>Dissolvenza - 10/07/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Come se un pensiero <br/> a lungo proiettato <br/> dalla mente su una tela <br/> vi rimanesse impresso <br/> così un ricordo <br/> reale nel cuore <br/> perso nel tempo <br/> rimango a mirare. <br/> Era palpabilmente fatta di sabbia <br/> di panna <br/> la mia donna. <br/> Nata dal nulla <br/> dei miei desideri. <br/> Regina di un impero di Sogni. <br/> E io, <br/> credermi capace, in diritto, di viverli. <br/> Qualche parola <br/> finalmente <br/> definisce i limiti <br/> del silenzio. <br/> Di lei rimane <br/> un alone digitale <br/> qualche istante <br/> sullo schermo. <br/> </p>
<h3><a name="3"></a>Eric Satine - Gymnopedies I - 09/07/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
C'era un tempo ineluttabile <br/> dove le nostre visioni <br/> erano ponti sul mare <br/> e i nostri sguardi, <br/> sicuri dell'impossibile, <br/> vi si incontravano a metà <br/> senza paura di guardar di sotto. <br/> </p>
<p>
	</p>
<p class="indent">
Poi il mio percepire s'è incurvato <br/> la speranza ingobbita <br/> è passata da freddi metallici occhi <br/> scrutando una città di nebbia <br/> il remoto gorgo che l'ha rapita. <br/> </p>
<p>
	</p>
<p class="indent">
E il pensiero che il suo sguardo <br/> non sia mai esistito <br/> m'ha sfiorato la guancia <br/> solitario graffio. <br/> </p>
<p>
	</p>
<p class="indent">
L'altra mattina ho compiuto un viaggio <br/> verso un'altra coscienza <br/> a piedi, come ginnastica. <br/> Ho viaggiato lungo i binari della sua melodia <br/> seguendo senza sapere <br/> immaginando senza volere <br/> al termine dell'Opera <br/> la risposta cercata. <br/> </p>
<p>
	</p>
<p class="indent">
Ho sentito serrarsi l'ostile diatriba <br/> allora chiara è apparsa <br/> la dicotomica contraddizione del mio concepire: <br/> sulla terra, sbagliando, ho cercato il riflesso del cielo <br/> ma intravedo il suo azzurro dentro pozze fangose. <br/> </p>
<p>
	</p>
<p class="indent">
E penso al vento, <br/> che sospinse le nubi <br/> che piovvero note ideali <br/> a descrivere lacrime dimenticate. <br/> Eccentrici sospiri di altri cuori, in altri tempi, <br/> ritornano a casa. <br/> Così, ora, non vedo più lei. <br/> Non posso non crederla <br/> Giulia <br/> mai più. </p>
<h3><a name="4"></a>Dialoghi e Monologhi - 05/07/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Un'eco lontana ritorna <br/> mi sorprende ancora <br/> senza un messaggio <br/> e vengo sospeso per un attimo <br/> sopra un luogo remoto <br/> familiare e lontano <br/> perché non c'è nessuno. <br/> </p>
<p class="indent">
Siamo in due a chiamare <br/> in fondo a questa rupe <br/> morbidi ricci, mano sotto il mento, <br/> giocando a pelota un muro risponde <br/> sempre <br/> quel che riceve <br/> senza vita. <br/> </p>
<p class="indent">
Mi si accosta un senso di vuoto <br/> è come morte apparente <br/> volontà suicida giace nel sudario <br/> né riccioli né volto <br/> ha quest'eco silenziosa. <br/> </p>
<p>
Potessi fare a meno di tutto questo nulla!</p>
<p class="indent">
Nulla più mi attendo <br/> quando la mano trova la calda carezza della terra <br/> son debellati i passati futuri <br/> tanti ponti interrotti, mai esistiti.  <br/> </p>
<p class="indent">
Ma c'è un pezzo di lei che mi batte nel petto, <br/> un contratto scaduto <br/> che nessuno ha firmato. <br/> Di quale paura segue lo spettro, <br/> a quale ombra s'appella <br/> sotto questo cielo di nuvole <br/> il suo silenzio? <br/> </p>
<p>
	</p>
<p class="indent">
Quale abilità in un cavalleresco gioco <br/> l'anello è infilato da un'amara lancia <br/> e dietro ancora <br/> me. <br/> </p>
<p>
	</p>
<p class="indent">
La mia richiesta onerosa <br/> un muto passare disattende: <br/> non più il corpo, non più <br/> l'anima o l'amore <br/> ma solo un attimo di attenzione <br/> in assenza di questi. <br/> </p>
<p>
	</p>
<p class="indent">
(su un treno per Milano <br/> in una pallida mattina come tante <br/> senza lei) <br/> </p>
<h3><a name="5"></a>Ritmico istante d'assenza - 25/04/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ecco, in giornate come questa <br/> sento le gocce del tempo battermi in testa. <br/> Ma questo suono arcaico, di savana <br/> quasi risonasse in un'umida cantina <br/> si distorce e perde il suo selvaggio richiamo <br/> alla vita. <br/> Ecco, le membra colpite si risvegliano <br/> ma non sono più mie, <br/> sono lontane, sono fantasma e aria fantastica. <br/> Ho sempre amato palpeggiare il vento <br/> e a quell'amore faccio ritorno <br/> senza più mani. <br/> </p>
<h3><a name="6"></a>Mi salva il cielo - 22/04/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Il tempo. <br/> Non è vero che scorre, il tempo. <br/> Esso rimbalza. <br/> Contraddizioni, illusioni, giorno dopo giorno, <br/> anno dopo anno <br/> avvolgono il tempo in una spirale cieca <br/> che procedendo verso l'infinito <br/> perde quei significati, scioglie quei nodi <br/> cui mi aggrappo. <br/> Avanti e indietro, ma su rette innumerevoli. <br/> Cado a pensarti remota nel mio futuro. <br/> Se il futuro è la strada, la vita, ben differente <br/> è il percorso. <br/> Mi salva il cielo <br/> dove ogni cosa si muove <br/> ma non importa in quale direzione. <br/> </p>
<h3><a name="7"></a>Sospiro - 21/04/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Era molto lento. <br/> Pallida alla vista nella penombra <br/> sciolta in un lungo brivido, un immane sussulto. <br/> Intorno a me tutto il resto era una capanna <br/> e sul tetto i suoi capelli ricci <br/> nei miei occhi, come volute di fumo. <br/> Ricordo il calore <br/> di un solo corpo senza soluzione di continuità, <br/> il soffice di una superficie assoluta, senza spigoli, <br/> concava nell'accogliermi, convessa nel protendersi <br/> a cercarmi. <br/> Era tutto molto lento, coinvolto, pieno. <br/> Questa notte ancora <br/> sarà il mio sogno. <br/> </p>
<p>
Volevo una casa - 03/03/2005</p>
<p>
Volevo una casa <br/> diversa <br/> la volevo costruire con te. <br/> </p>
<p class="indent">
Ricordi quando ragionavamo dei colori? <br/> Pareti variopinte e stanze rosse di passione <br/> con angoli azzurri di sereno silenzio. <br/> </p>
<p class="indent">
C'è un posto dove nessuno ti giudica. <br/> Dice che c'è un luogo in cui non si distinguono <br/> coraggio e pazienza. <br/> Ne volevo fare la nostra casa,  <br/> non appena l'avessi trovato. <br/> </p>
<p class="indent">
Sarei stato seduto per terra <br/> immaginando letti e poltrone <br/> e avrei goduto la vista dei tulipani nel giardino <br/> che avresti immaginato tu. <br/> </p>
<p class="indent">
"La protezione non sta nel tetto, <br/> ma nel calore di una reciproca comprensione". <br/> "La libertà non ha bisogno di porte e finestre, <br/> vive dove si guarda con cuore maturo, <br/> dove si sceglie cosa guardare, attimo dopo attimo". <br/> </p>
<p class="indent">
Piano...piano... <br/> per non svegliare il bimbo che sogna <br/> che cresce. <br/> Dove cresce? <br/> </p>
<p class="indent">
Avevo piantato un seme, <br/> forse volevi una casa sull'albero. <br/> Per darti una casa nel vento <br/> volava un aquilone. <br/> E sulla spiaggia del tuo mare <br/> ti avrei costruito un castello. <br/> Ma dov'è che abita l'amore? <br/> </p>
<h3><a name="8"></a>La risacca dopo la tempesta - 03/03/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Se il mio cuore potesse vederti come un'onda <br/> dell'immenso oceano che mi circonda <br/> nuoterei in ogni attimo sinceramente sereno <br/> per riferirtelo domani. <br/> Ma tu non sei soltanto un'onda, <br/> sei la gioia, l'amore, la rabbia <br/> l'attesa <br/> e non lo sai. <br/> Sei tutto ciò che anche oggi s'è infranto <br/> sulle mie sponde. <br/> </p>
<h3><a name="9"></a>Freddo intenso - 02/03/2005</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
In questa notte di ghiaccio <br/> la luna ha un anello <br/> che bianco argento pende nel buio. <br/> C'è un'aura di gelo <br/> sulla cresta dei monti <br/> ancora più alti, placidi, silenti. <br/> Indifferenti. <br/> </p>
<p>
	</p>
<p class="indent">
Mi piove addosso la luce sintetica dei lampioni <br/> che attenderanno un altro giorno in piedi <br/> quando la mia coscienza sarà già evaporata. <br/> Passando da un'alba all'altra, <br/> da una luce ad un bagliore <br/> mi vien da chiudermi dietro <br/> la porta di questo giorno. <br/> </p>
<p class="indent">
Mi si storpia davanti un futuro <br/> di stanze bianche, gelate, alcune vuote <br/> e i miei passi, ora proiettati nel nulla, <br/> riempiono un corridoio di infinite porte. <br/> C'è un'ansia crescente <br/> per gli usci che alle mie spalle si serrano. <br/> Troppe speranze abitano la prossima stanza. <br/> Troppe la precedente. <br/> </p>
<p>
	</p>
<p class="indent">
Sarà forse così, sempre più così <br/> questo mio passeggio? <br/> Forse la vita finisce con la curiosità. <br/> </p>
<h3><a name="10"></a>Universo - Agosto 2004 - Novembre 2004</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Amarsi semplifica l'universo. <br/> Sciolgo le vele, raccolgo la cima tagliata di un'ancora adagiata sul fondo. <br/> Parto. <br/> Il mare prima era una tavola immensa <br/> Sulla quale far scivolare le nostre vite come sul biliardo. <br/> Buca. <br/> Navigo in 4 dimensioni, Ulisse dello spazio e del tempo <br/> Ritorno ad una sconosciuta Penelope. <br/> Il mondo è grande, il tempo è di gomma, dove sei, Penelope, <br/> Dove. <br/> In quale epoca sei stata, in quale luogo a me irraggiungibile. <br/> Quante isole segnate sulla mia mappa, quanti scogli nascosti <br/> Come siluri a squarciare lo scafo. <br/> Eppure io ti ho già trovata,  <br/> navigando nel caso delle 4 dimensioni. <br/> O forse eri Circe. <br/> So che esisti, che la mia isola è fatta di terra, <br/> che quella terra è la mia Patria. <br/> Facile allora dimenticare le dimensioni del viaggio, <br/> la sua natura, il suo centro <br/> Il suo arrivo. <br/> In balia di onde quiete lasci la barca per toccare con mano <br/> quel che non puoi toccare, se non sei fatto per trovare. <br/> Sei solo <br/> In un universo parallelo. <br/> </p>
<h3><a name="11"></a>Scale mobili</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Una nonna col bimbo <br/> al parco. <br/> L'alba in un occhio <br/> il tramonto nell'altro. <br/> </p>
<p>
	</p>
<p class="indent">
La parabola dell'Astro <br/> è una nostra illusione, <br/> dato che quel che non c'è non esiste, <br/> di notte dimentichiamo pure il sole. <br/> </p>
<p>
	</p>
<p class="indent">
Ho visto una persona salire le scale. <br/> Ho visto due persone scendere le scale. <br/> Non ho visto altri più. <br/> </p>
<p>
	</p>
<p class="indent">
Mentre c'è chi racconta parabole  <br/> dove la vita è fatta a scale. <br/> C'è chi scende e c'è chi sale. <br/> O ellissi che descrivono la vita. <br/> Come una ruota. <br/> </p>
<p>
Nonna, cosa mi racconti oggi? <br/> Nonna?</p>
<h3><a name="12"></a>Roma</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Tu non ricordi Roma <br/> L'ho rivista passare distratta <br/> Con i suoi occhi sudati di inizio settembre <br/> Che ti cancellano la voglia di scherzare <br/> Il pensiero che ci sia qualcosa di importante <br/> A cui tornare. <br/> Se le sue eterne rovine vorranno aspettare <br/> Un'altra rovina potranno contare tra loro <br/> Eterna anch'essa suo malgrado <br/> Anch'essa figlia in qualche modo di Roma <br/> Attaccata alle sue infinite mammelle <br/> alle notti di transito  <br/> tra le sue braccia di prostituta che ti aspetta, <br/> alle mani unite  <br/> tra gli aghi dei suoi pini. <br/> </p>
<p>
	</p>
<p class="indent">
Lei che non ti da un attimo di tregua <br/> Che ti ricorda sempre dove sei e ti chiede dove stai andando <br/> Anima così persa, così piccola, che non si sa come fa a vederti. <br/> Nonostante tutto. <br/> </p>
<p>
	</p>
<p class="indent">
La forza con cui occhi di marmo mi chiedono di te <br/> la debolezza di un sospiro <br/> in risposta. <br/> </p>
<p>
	</p>
<p class="indent">
Scende la sera a commuovermi, <br/> scivola nell'orecchio un canto di solitudine  <br/> una moltitudine di soli, <br/> potrebbe illuminare infiniti tramonti. <br/> Il cielo invece si spegne. <br/> </p>
<h3><a name="13"></a>Parole d'odio verso il mio Amore</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Le tue foto in cui mi abbracci <br/> I mille dolci sorrisi d'amore <br/> Saranno frecce avvelenate che instillano odio. <br/> </p>
<p>
	</p>
<p class="indent">
L'amore mio è chiuso in gabbia  <br/> Da una sconosciuta sotto mentite spoglie <br/> Che ritaglia una sagoma sempre più stretta <br/> Per mascherarmi da pagliaccio  <br/> Che cinge non più le braccia al collo ma alla vita <br/> Stringendo, si che la vita sia soffocata <br/> Che mesce acqua e fango  <br/> Per dimenticare il sapore del vino. <br/> </p>
<p>
	</p>
<p class="indent">
Alzo gli scudi contro queste ed altre infinite frecce, <br/> come una volta promisi a me stesso di difendere questo amore <br/> da chiunque <br/> ora rinnovo la promessa scoprendoti nemico <br/> freddo, implacabile e falso. <br/> Alzerò la spada, ma fenderò solo l'aria della sconfitta, <br/> tu sarai lontana, dietro foglia di fico o di ulivo. <br/> </p>
<p class="indent">
Odiare, trasformare la passione errabonda e accecata <br/> coltivare la piantina nel concime dell'abbandono <br/> per riuscire a cogliere domani i tuoi freddi frutti dell'indifferenza. <br/> 	 </p>
<h3><a name="14"></a>Un gatto a Cittàducale</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Ho visto un gatto <br/> grigio <br/> rasentare un muro di una casa <br/> gialla <br/> una pietra con segni di un tempo remoto <br/> e poi... <br/> pure là ho trovato che mancavi. <br/> </p>
<h3><a name="15"></a>Addio</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Gocce bianche <br/> 	fredde <br/> 	dalla punta delle tue mani. <br/> 	Mi cadono negli occhi, mi accecano. <br/> 	Un universo è dimenticato <br/> 	un universo viene abortito.</p>
<p class="indent">
Così comincia una lunga corsa <br/> 	al di là della nebbia <br/> 	e lascio la nebbia per il fumo.</p>
<p class="indent">
Se almeno fosse il piano di un criminale... <br/> 	ma è solo la tua inconsapevolezza, <br/> 	una tenera colpevolezza per un subdolo delitto.</p>
<p class="indent">
Quale pace mi dona la dignità del silenzio <br/> 	e che vergogna invece l'abbandono <br/> 	al quale mi abbandono.</p>
<p class="indent">
Ancora sale la voglia di crederti <br/> 	ma non vorrei più <br/> 	sentirmi in bocca il rigurgito mentale <br/> 	di un pasto troppo bello.</p>
<p class="indent">
Sono padre di un innocente <br/> 	non più accettato, non più capito. <br/> 	Per salvarlo lo affido al destino. <br/> 	Non a uno Spedale Fiorentino <br/> 	ma a una parola fredda come la lama <br/> 	chiusa come una porta <br/> 	metto in braccio il mio Amore.</p>
<p>
Addio.</p>
<h3><a name="16"></a>Barboni alla stazione- 22/09/2003</h3>
<p class="returnIndex"><a href="#index">[indice]</a></p>
<p class="indent">
Nell'abbraccio di una coperta <br/> 	il sipario <br/> 	e dietro, al di là dei nostri sguardi <br/> 	la vita. <br/> 	Dai buchi nei pantaloni <br/> 	come occhi <br/> 	albeggia un crepuscolo <br/> 	ignorato a forza d'abitudine <br/> 	dalla bieca luce fredda <br/> 	neon di esistenza normale.</p>
<p class="indent">
Le porte di un altro treno <br/> 	si chiudono. <br/> 	Piedi rognosi <br/> 	già immobili <br/> 	si fermano. <br/> 	Passeggia la notte ormai per le strade. <br/> 	Sono quasi arrivato nella mia convenzione <br/> 	dimora stamberga <br/> 	calda illusione. <br/> 	Sono salvo, <br/> 	ma tendo ancora l'orecchio <br/> 	ai silenzi di coloro che viaggiano <br/> 	senza partenza, <br/> 	sotto l'abbraccio della coperta.</p>
<h3><a name="17"></a>Ricerca Siderale - 2002</h3>
